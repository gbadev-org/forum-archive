The counterpart to Super NES mode 7 is GBA mode 2. This looks like it might be an attempt at real bug-for-bug compatibility with undocumented GBA graphics hardware modes numbered 6 and 7:
  • mode 0: 4 text bgs
  • mode 1: 2 text bgs and 1 rot/scale bg
  • mode 2: 2 rot/scale bgs
  • mode 3: one 16-bit 240x160 bitmap
  • mode 4: one 8-bit 240x160 bitmap selectable from two pages
  • mode 5: one 16-bit 160x128 bitmap selectable from two pages
  • mode 6: ???
  • mode 7: ???
