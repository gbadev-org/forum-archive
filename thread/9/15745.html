#159397 - eKid - Mon Jun 30, 2008 5:33 am

I use grit.

Using a GUI interface to convert graphics can sometimes be time consuming since you have to manually convert the graphics every time you edit them. Then after converting you must replace the old source files, rebuild...etc.

It's easy to setup the makefile to convert graphics automatically (using grit). There's a thread somewhere that explains how to use grit with the devkitPro template makefiles, I'm not sure where it is though....
