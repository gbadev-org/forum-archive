#7265 - NovusAmor - Fri Jun 13, 2003 1:41 pm

No it can't export to C/C++ GBA format (source code). It however does allow you to set your color bit-depth. Thus you can make all your palettes use only 5 bits per color (GBA color format) so you can see what it should look like on the GBA. It also has a bunch of other very useful functions like setting up grid locking so you can create images using tiles (8x8 blocks), creating animations, ect.

I would highly suggest getting Pro-motion if you're serious about making graphics for the GBA. I think it's pretty cheap, plus there's also a trial version if I remember correctly.
