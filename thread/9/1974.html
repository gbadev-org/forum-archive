#10227 - Nessie - Sat Aug 30, 2003 2:34 pm

1) The Tile View window was one of the first things that I did, so no, there is no way to scroll around it. It's actually something that has annoyed me a bit, usually I just use smaller tilesets, like 128x64 and/or zoom out to either 2x or 3x. So, yes, this is something I will try to fix for the next release if it doesn't take too long. Otherwise it'll have to wait until the release after that.

2) Hmmm, the un-lite (full) version allows a much larger selection. I'll double the number for the next release, but in the meantime, one thing that might help (depending on what you are trying to do) is press 'T' to trim out the transparent tiles. If you have a lot of transparent tiles in the selection, that will make the selection smaller, meaning you can then add additional items. Sort of lame for now, but...

3) Any feature that is greyed out is either unavailable in the un-lite version or has not been done yet. In the case of the import layer from, that will be available in the lite version, possibly the next release or definetly by the one after that.

Cheers
