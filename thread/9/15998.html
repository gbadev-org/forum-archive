Hi Adson, welcome to the forums. In general, please start a new thread for your questions instead of resurrecting an old one. (Mods, if any are still around, is it possible to break this into a new thread?)

adson3492 wrote:
Hi guys, I was wondering if there was any way to make certain parts of a BG transparent while the others are opaque(For GBA in case you're wondering what system)?


Yes. Color 0 of your palette is always rendered as transparent. So any BG tiles that use color 0 will have transparency. I'd recommend you read up on how color palettes work, you can refer to this page on tonc, and if you haven't read the beginner's FAQ here, please do.
