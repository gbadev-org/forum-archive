Key_46 wrote:
This can sound a stupid question but...what is the difference betwen a PC and a DS?...both have WiFi!

A PC running "soft AP" software can act as a Wi-Fi access point. A DS running software that uses the DSWifi library cannot. If you can code, and you want to add soft AP functionality to DSWifi, go ahead and talk to Stephen Stair, maintainer of DSWifi.
