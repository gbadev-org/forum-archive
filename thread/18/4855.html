Is there any way to get to the ROM or RAM on the DS?
What is a romdump?

#34346 - netdroid9 - Tue Jan 18, 2005 10:00 am

Short answer: No

Long answer: Yes, but no.

Full answer: You can access it using the GBA mode, but it's limited to that of the GBA. A bootloader for the DS that reroutes code to the GBA port for execution has been made by Darkfadar/Darkain (Can't remember which at the moment, but I think it was Darkfadar. Darkain is working on the NiFi Auth data).

A romdump is a dump of a rom executable, also known as a rom.

Ever used an emulator? To use those you input a romdump.
Ever compiled code for an embedded device (Example: GBA/GBC/GB)? The output was the equivilant of a romdump.
