#110851 - dantheman - Sat Dec 02, 2006 6:44 am

Well, for one thing, this isn't hacking. It's homebrew. Running non-official code on the DS system.

In any case, I believe that you can indeed use your Nintendo Wifi USB connector as a softAP for use with homebrew programs that use Wifi, as a replacement for a wireless router, but this requires the use of Linux as far as I know.

This would allow you to use WMB, or Wireless MultiBoot. This sends a signed DS demo to the DS via DS Download Play. Note that if you flash your firmware using FlashMe, you can send unsigned DS programs up to 4 MB in size as well. Installing FlashMe however involves the use of a passthrough device. WifiMe is a virtual passthrough device with which you can install FlashMe. In both cases, you still need something to store the flashme.nds file on, like a GBA flash cart, Supercard, M3, or other homebrew device.

However, the ability to use WifiMe, aside from having the correct wireless chipset, is mostly dependent on your DS firmware version. Search the forums here for directions on how to use the TakeMeOut method to test which version you have. If your DS is version 4 or above, you cannot use WifiMe, and you must therefore purchase a different passthrough device like a NoPass variant.
