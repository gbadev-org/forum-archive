Are you sure you have to create a bridge? If not,
try ICS/Internet Connection Sharing. Or does the software do that automatically?It looks like it is auto... Is it forcing you to only use bridges?
Can you enable SoftAP, without it trying to bridge?

Here's some instructions for ICS. It's from the instructions for installing
the N dongle homebrew drivers and software. Enable SoftAP in your wireless card's configuration software. This time, instead of bridging,
follow these instructions.
Quote:

Now go to Control Panel -> Network Connections. Right-click on the Network Adapter you are currently accessing your Internet through and open the properties for that connection. Go to the Advanced tab, and click "Allow other network users to connect through this computer's Internet connection". Then click OK. If you get an error message saying the IP address is already in use, disable the Network adapter you are trying to share then try again - be sure to enable it again afterwards. It can take a long time for ICS to be enabled so wait patiently.
