I'm assuming you mean "What do I need to flash my firmware?"

Ok, first you'll need either a *PassMe* or a Wifi card compatible with *WifiMe*, and a GBA cat, or as of...today, a GB Movie Player with hacked firmware.

The *PassMe* is probably the more straightforward way right off the bat. Basically it's a hardware passthrough device...needs a commercial DS game inserted into it to work. It tricks the DS into running DS code from the GBA slot instead of the NDS card. If you flash the NDS code to the GBA cart, look for an NDS.GBA file (or if using a hacked GBMP if I understand how it works right it would load the code, in this case having the extension NDS from the root directory of your compact flash card), then when you start up the DS with a PassMe inserted, it'll boot that code...so if it's the flashing program, get ready to start flashing, and read the instructions on FlashMe's Homepage carefully. You'll need a paper clip or something to short that connector, and make sure you are good with the power...if you don't at least finish flashing the secure part of the firmware, you just turned your DS into a nice looking coaster.

Ok, so your other option is *WiFiMe*. Basically this is like a virtual PassMe, it works the same way. Go here for more information. You'll need the custom drivers and the WiFiMe data located there...also, it works only with specific wireless cards, this driver. Look at the compatibility list on the download page...you're in luck if you have one of those cards, then you can get started right away.

After you flash your firmware, you can also use this to send DS homebrew straight to your NDS via the download play option...which is quite cool. :)

Oh yeah, and another thing...the difference in an NDS file and an NDS.GBA file is this...

The NDS file uses the standard NDS format...good for sending via WiFi and in Darkain's Multi-NDS loader. You'll find all official demos in this format, although it's easy to conver them to the other. The NDS.GBA is this with a loader wrapped around it making it possible to use from a GBA cart using PassMe...it even has a GBA header so that the flashing programs can muck with it all they want and not cause a fuss. You can find this loader on Dark Fader's website. Look for NDSmall.bin.

To run DS homebrew on a Nintendo DS, you'll need either a compatible Wi-Fi card (which costs money) or a PassMe adapter and a flash card (which costs money). If you want to use Wi-Fi to run homebrew, and you only want to load homebrew when you're next to your PC (and possibly keep it in sleep mode elsewhere), then you'll need FlashMe. If you don't have a flash card, you'll need to find somebody else who is willing to flash your DS, either through the FlashMe Meetups thread or through the paid service that was advertised in some thread either here or on Pocket Heaven.
