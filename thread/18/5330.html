You will only be able to do this with Mark Overmars' help, because:

- Reverse engineering the files violates the GameMaker usage license
- The game files are encrypted before they are added to the runner

It would also be a huge effort to create the DS runner application, as the PC application is tied deeply with DirectX. You would have to write everything from scratch. It would end up taking years for a small group to complete, the only way this would work, is if Mark Overmars agreed to sell a license to a large commercial group who could work fulltime on this. And it really isn't worth that much effort. If you're too lazy to learn C or C++ and get coding for yourself, then fuck off.
