#176735 - dantheman - Mon Sep 26, 2011 7:32 pm

English again, sorry. This is from my limited knowledge, which is probably wrong since I've been out of the DS loop for ages:

There are many flash carts that can access DS mode on the 3DS and DSi. They work by reflashing the firmware to make themselves appear like a legitimate game (faking the header, for instance). The CycloDS cart is the only one that can run DSi stuff in DSi mode. There are currently no flash carts that can run code using the 3DS's upgraded hardware.

The latest homebrew programs, compiled with the latest version of DevKitARM, can use the DSi's 16 MB of RAM and maybe the faster CPU. I would assume that these homebrew programs would require either SudokuHax or the CycloDS cart that can run DSi code, but I honestly don't know for sure.

Unfortunately I too do not know where any 3DS hacking discussion might be taking place. GBAdev here is, well, kinda dead.
