1) Download and install Xcode
2) Download and install devkitARM (both Intel & PPC available).

Additionally, it's good to download the nds examples package to see if your installation is working. If you're using different user for installing and actual programming you may need to set DEVKITPRO and DEVKITARM to your environment pointing to correct directories.
