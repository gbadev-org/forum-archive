#158317 - strager - Mon Jun 09, 2008 3:48 am

Try pre-pending a period (.) before local symbols, e.g.:
Code:

; myFunc
;   Does nothing.
myFunc:
    mov r0, #5
.loop:
    subs r0
    bnz .loop

    bx lr


This helps prevent naming conflicts, and make code a bit more understandable.
