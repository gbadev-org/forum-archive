doublec did several really good tutorials for beginners/intermediates:
http://forum.gbadev.org/viewtopic.php?t=5757

FlashMe has removed RSA signature checks so Firefly's wmb.exe can load not one but multiple roms simultaneously for DS Download Play without the flash cart. So, flash carts aren't as necessary once FlashMe has been installed (mine are collecting dust).
http://ds.gcdev.com/dsfirmware/

Omalone is widely considered the best homebrew game release for the DS so far: http://omalone.free.fr/index.html

Lynx started a homebrew rom repository at: http://www.ndshb.com/

There's a fair amount of commercial demos out now. There is a sticky topic in DS Flash Equipment but somebody ignored it and started a new thread which might be more up-to-date (I haven't checked):
http://forum.gbadev.org/viewtopic.php?t=6005&start=0

Darkain has done work on a mutlirom booter based on Darkstar. There's a package of the commercial demos all in one rom and there's the updated utility he submitted to the GBAX compo.
http://forum.gbadev.org/viewtopic.php?t=5732&start=0

That's the quick and easy of it, enjoy.
