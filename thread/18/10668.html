The things I did to get my D-Link DI-524 (E) to work (I have it plugged to my ADSL router), hope this is of any help:

* WEP 128bit HEX, used a 26 HEX characters long password (e.g., 0 pad it, if you must)
* Set DI-524 to work as a DHCP (don't know if it can be used just a passthrough device,
seems like not) so DS would be in its local network, getting an IP like 192.168.0.x
* Opened up port 8888 for incoming/outgoing traffic, because DI-524 blocks almost all ports
by default

So I use port 8888 for homebrew stuff, but Mario Kart would connect without opening any specific ports. Perhaps it uses a port like http that is open by default, etc...
