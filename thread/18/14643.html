#146749 - simonjhall - Sat Dec 08, 2007 2:34 pm

Setting the subscreen to MODE_1_3D make both screens show 3D? Really? Is this on the real kit or is this in an emulator?
Doing 3D on two screens is something that is a little involved. Look up 'capture' with the search function and I'm sure you'll turn up loads of "3D on two screens" posts.
If you just want 3D on one screen and want to choose which screen it is, set the main graphics controller to use 3D as you've done and then use either lcdMainOnTop() or lcdMainOnBottom() to choose which physical screen it's on.
