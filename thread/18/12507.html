I'd like to use Bank E&F&G for main sprites memory ( 64Ko+16Ko+16Ko )
Is there a way to do that?

#119217 - Sausage Boy - Tue Feb 20, 2007 9:59 pm

Whenever I want to find out something like this, I consult dovoto's Large intimidating VRAM table.

As you can see,
bank E can be mapped to 0x06400000 as Main Sprite Graphics,
bank F can be mapped to 0x06410000 as Main Sprite and
bank G can be mapped to 0x06414000 as Main Sprite.

So yes, it seems like what you want is possible.
