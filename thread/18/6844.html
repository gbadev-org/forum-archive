So, to that end, if rygar wants to use the Programmers Notepad to compile, he'll have to set up a couple of custom tools for make and clean.

First, open the "Tools" menu and select "Options".

Then, select "Tools" in the left tree view of the Options window.

Now, select the "(None - Global Tools)" scheme.

Then, click the "Add" button.
