#38118 - tepples - Tue Mar 22, 2005 3:04 am

GBA games that run on a GBA SP run unchanged in single-player mode on a Nintendo DS.

GBC games do not run on a Nintendo DS. GB mono games need an emulator. Early tilt-sensor games for the GBA do not run on an SP or a DS because their Game Pak slot is oriented backwards compared to that of the original model, but newer games such as WarioWare Twisted compensate for this.
