#42833 - oxymen - Sun May 15, 2005 12:33 pm

oops, I meant the tool dsbuild.exe.
navigate to the folder you have dsbuild.exe in with the command line.
type dsbuild indata.nds outdata.nds.gba
exchange the indata with the name of the rom you want to convert and outdata to the name you want on the converted file. remember to have the file you want to convert in the same folder as dsbuild.exe

#42846 - maniacdevnull - Sun May 15, 2005 5:02 pm

Godster, you also need to make sure you have the multiROM mode turned off. Some early versions of the linker software don't allow that, so make sure you get the latest version from here: http://www.ezf-advance.com/download.htm. It should be the third item on that page. Once you have it, use the linker to erase everything from the cart, click the swiss army knife button, and uncheck the multiROM option. Once you do that, you should be able to burn one .nds.gba file to the cart and run it using [pass|wifi|flash]me.

HTH
