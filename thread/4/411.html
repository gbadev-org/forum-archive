#2598 - tepples - Sat Feb 08, 2003 4:28 pm

What kind of "ROM editor" do you mean? A tile editor? An audio editor? By "Mac" do you mean Mac OS 8/9 or Mac OS X?

If you mean a tile editor, you can download Pin Eight NES Tools which, despite the name, contains a tile editor that works with several target platforms' formats including GBA 4-bit and 8-bit. It uses the Allegro library, which works on Mac OS 9 and Mac OS X.

If you mean an audio editor, it should be simple to write a program that converts a game's unsigned samples to and from AIFF.
