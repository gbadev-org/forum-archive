#6866 - Jason Wilkins - Tue Jun 03, 2003 8:22 pm

Edit cmd-rc.bat and make sure that the variable DEVKITADV is set to the directory where you unziped the devkit.

Then, there is a shortcut, called DevKit-Advance-R5-Beta-3.lnk which you can click on and get a command line shell which is setup and ready to go.

I use notepad and the command line most of the time, so I can't help you set up an IDE ^_^
