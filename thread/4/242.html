They don't take demos in this competition. But if you're a demo coder at heart, you can still enter this competition:

1. Write a simple rhythm game or puzzle game that uses only a few sprites.
2. Write a demo that runs in the game's background. Make sure you have some good music.
3. Put the game on top of the demo.

I got this idea when I realized that the backgrounds for Dance Dance Revolution look sort of like a demo.
