bit 0: 0x0001
bit 1: 0x0002
bit 2: 0x0004
bit 3: 0x0008
bit 4: 0x0010
bit 5: 0x0020
bit 6: 0x0040
(in general) bit N: (1 << N)

A with bit 0 set: A | 0x0001
A with bit 3 clear: A & ~0x0008
A with bit N flipped: A ^ (1 << N)
