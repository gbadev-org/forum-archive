Announcements And Comments > Handheld Dev Env v0.1

#55001 - NoMis - Sat Sep 24, 2005 12:39 pm

Hi everyone,

I released the first version of my Handheld Dev Env plugins for eclipse today. The plugins try to simplify homebrew development of handheld games by extending Eclipse and the CDT (C/C++ Development Toolkit) plugins.

For now there are plugins for GBA development providing you with managed makefile build that extend the CDT Managed Build System to provide easy creation of GBA projects as well as Launch configurations that automatically start debugging with VBA-SDL within the workbench.
An Online help with guides to the HDE features and lot's of usefull links to GBA hombrew dev sites.

More information is provided on the HDE Homepage.

I will also use our Mantis page to document future enheancements. This is also the place where you can make feature request or bug reports.
