#158897 - ole - Fri Jun 20, 2008 11:41 am

I just want to announce that I have ported part of the Pstros MIDP implementation to the NDS. It means that you can run java programs for the mobile phones on the NDS. It runs upon the CLDC java machine from Sun (KVM) that was originaly ported to GBA by Torlus and lately to NDS by davr. More info and the binary is here: http://www.volny.cz/molej/pstrosnds/
