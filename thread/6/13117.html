BITS present their first demo for GBA:

BITS #2000 "Colors"

It is a color and sound test for Mode3.
It was written with "fasmarm", Linux,
Visual Game Boy emulator for testing
and gbafix for the cartridge header.
It is less than 512Bytes in ARM code
(the cartridge header is 192Bytes).

v20070501
