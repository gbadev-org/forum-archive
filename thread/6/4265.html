Announcements And Comments > JPEG Viewer Pogoshell Plug-in (with source)

#27902 - gb_feedback - Sat Oct 23, 2004 10:52 am

As I promised to some people some time ago, I have put up the source of this JPEG viewer Pogoshell plug-in on my site.

The hard part is totally based on Burton Radons' excellent public domain JPEG code, with a couple of minor alterations. The program is written as multiboot and then compressed to a .mbz file which is 14k bytes. For low battery use, it also uses sleep mode after displaying the picture, waking up every VBLANK to check if any key has been pressed.

The code is a copy of my folder which is set up to use devkitadv, via Visual Studio 6, so in theory some people may be able to run with it for improvements. Please let me know if there are any...
