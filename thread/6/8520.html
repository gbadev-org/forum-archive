#72020 - tepples - Thu Feb 16, 2006 12:35 am

If you find any GBA program under GNU GPL or any other free software license, you already have the OK to distribute copies for a fee as long as you provide complete corresponding source code on the disc. TOD is such a program, and so are most of the rest of my programs.

Watch out: some programs are free but come aggregated with unfree assets. Luminesweeper can be free if you make a set of free skins. The same applies to a slideshow program built around Burton Radons' permissively licensed JPEG decoder.

And if you need to pad the disc a bit more, try emulators. As PocketNES is GPL[1], PocketNES aggregated with redistributable NES ROMs is also redistributable. You can raid the pdroms.de collection for NES ROMs with acceptable licenses.

[1] All of PocketNES except the saved state compression code is permissively licensed.
