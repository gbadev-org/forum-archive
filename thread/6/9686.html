Announcements And Comments > New Version of VBAstep (Mac Debugger)

#84770 - ribrdb2 - Wed May 24, 2006 5:50 pm

Just released, VBAstep 0.3a, now a Universal Binary. New features include an OAM memory viewer, fixed gdb write watchpoints, and new gdb breakpoint code, so debugging should now work properly with any version of gdb and gcc. Still missing is sound and linux support. Let me know if you want these, or anything else. If people are using it I'm more likely work on it.
