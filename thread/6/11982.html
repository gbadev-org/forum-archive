Free Pascal compiler for NDS is now available. You can download it from my web page (it is about 3.6 MB). I have started converting libnds headers to pascal and, some bugs and some "to do" apart, seems that it works very well. In the next few days I'll provide to translate some examples too. :)
As for gba, this compiler uses binutils and tools from devkitPro so feel free to make a donation to them!
