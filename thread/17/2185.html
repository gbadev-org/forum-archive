Sometimes I get the same thing with official Game Paks and have been getting it since the days of the 8-bit Nintendo Game Brick in the early 1990s.

Problem: GBA isn't reading header from Game Pak
Symptom: No "Nintendo" logo or scrambled "Nintendo" logo
Cause: Bad connection between GBA and Game Pak
Solution 1: Reseat Game Pak in GBA by removing and reinserting it
Solution 2: Clean GBA and Game Pak edge connectors with Nintendo's Game Boy Cleaning Kit
