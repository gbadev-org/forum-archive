#91785 - tepples - Sun Jul 09, 2006 4:02 pm

The number of GBA games that you can put on a SuperCard is limited only by the size of your SD card. No GBA game is bigger than 32 megabytes, and most homebrew games (from gbadev.org, pdroms.de, etc.) are 2 MB or smaller (and can be much smaller). For instance, TOD (a soviet-mind-game implementation) is only 160 KB.

SuperPass 2 is a device that lets one run Nintendo DS code from the SuperCard. Nowadays, you'd want the SuperKey instead, which does the same thing but more cleanly. (SuperPass 2 is a PassMe2, while SuperKey is a NoPass.)

SD, or Secure Digital, is a form of flash memory used in many digital cameras. CF, or CompactFlash, is another. Any electronics store should carry them, or you can sometimes find great deals on eBay. You have to choose the one that matches the SuperCard you buy.
