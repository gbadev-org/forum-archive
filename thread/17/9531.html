#82846 - Ultima2876 - Wed May 10, 2006 7:22 pm

As far as I know, any flash cart will do all of that. However - the flash cart you use doesn't effect how the emulators run - every GBA is the same (or roughly so anyway - you could talk technicalities and say they all have ever so slightly different processor clocks, but eh), and since the flash cart doesn't effect emulator performance, it means that emulators will run how they run. That means the SNES emulator doesn't really run very playably, etc.

You can expect NES games to run well, and master system/game gear games run really well too.

I don't have experience with any of the other stuff, but I assume that the same kinds of rules apply.
