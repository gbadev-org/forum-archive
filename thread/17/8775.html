For Windows XP

1 - Plug int the Cable from into the USB port of your Computer
2 - It recognize it as a Usb <-> Serial Port
3 - Int the Hardware Update Wizard choose
3.1 - Install from a list or specific location -
> then next
3.2 - Don't search, i will choose the driver myself-> then next
3.3 - Then choose - Show All devices -> then next ( this takes a while and it looks like the computers is locked, but be patient)
3.4 - Click Have Disk and choose the location of the drivers
3.5 - It will display EZF Advance Cable -> then next (it may complain about signatures, but proceed anyway) untill it finishes
3.6 - Turn on the gba pressing select + start untill the nintendo logo disappears or all that remains on the screen is the Gameboy letters The driver is installed on the device manager under - PCMCIA and Flash Memory devices

4 - If you get the words EZFA Cable not found
go to go the device manager on the - Universal serial bus Controllers You'll find a - High speed Serial Converter(this is a generic driver installed by microsoft) Change it's driver by clicking Update Driver. Then the Hardware Update wizard will appear
4.1 - Install from a list or specific location -> then next
4.2 - Don't search, i will choose the driver myself-> then next
4.3 - Click Have Disk and choose the location of the drivers
4.4 - It will display EZF Advance Cable and the other generic drivers from microsoft choose the EZF Advance cable the click next untill it finishes
4.5 - Turn on the gba pressing select + start untill the nintendo logo disappears or all that remains on the screen is the Gameboy letters
