#66470 - MrD - Wed Jan 11, 2006 1:25 am

It's odd that a graphics mode wouldn't work while using the cable.
An ezfa user myself, I can only think of a few things that would stop you using the ezfa for multiboot:
- You didn't set the linker up for creating a multiboot rom (there's different ways of doing this with int's or defines)
- You didn't code your ROM correctly (there are some discrepancies between some emulators and hardware)
- You're not using the correct ezfa client for your cart. Afaik, you can't use any other software for multiboot except the EZFA Client by Borden.
- Your rom is too big for multiboot
- Your gba is busted in a very odd way.
