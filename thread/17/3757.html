#23830 - cochese - Thu Jul 22, 2004 2:57 pm

I've just solved the problem. There's a new version of Power Writer .... v2.20 with that I could overwrite all again and it works like before.
I've entered the card size (512MB) in the config settings and didn't entered any boot rom so the program would use it's internal.
Phew! Glad my card is back! :)
