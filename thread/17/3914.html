Check this page out
http://www.canerdian.ca/gba.html

especially this file
http://www.canerdian.ca/_graphics/ezfa/Manual_EZF_Advance_(EZFA).pdf

I use the EZFA, and it let me backup my games to my PC, which can then be played in the VBA. The PDF shows step by step how it is done. I found it very easy.

I have also used the EZFA to run my own code that I found in tutorials on this site. The best feature in my opinion is that I have not taken the EZFA cartridge out of my gameboy in probably 3 months, as everything it does is through the link port at the back of the gameboy.

The software could be more user friendly in my opinion, but otherwise I am very satisfied with this product.

Also, this site http://bbs.gba-toys.com/ is the help forum. You can ask others questions there.
