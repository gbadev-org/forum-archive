#18661 - Mill - Wed Mar 31, 2004 3:11 am

You can't use the F2A Ultra 512/1G with the GB Bridge.
But F2A Ultra is compatible with the old F2A. So if you use the old
writer program(F2A Writer 3.2) and write GB/GBC files with it, you can
use the cart with the GB Bridge. The only drawback is, you can only
use 256 MB of the whole cartridge.

So, theoretically, it's possible that someday the new F2A Power
Writer might support the GB Bridge one way or the other. But
for now you have to treat it as an old F2A if you want to use the GB
Bridge.
