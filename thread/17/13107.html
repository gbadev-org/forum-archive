I'm not really sure what to tell you. I have had no issues with my F2AU cart, so I haven't really had to deal with Linker4U's customer service. Yeah, I meant Select+A when linked up, sorry about that. I didn't have my F2AU with me, so I didn't check it. Regardless, when you enter the Select-A menu, there should be some info on the top that tells you which buttons to press to access various functions. On a side note, you were using winf2a12.gba? Holy crap that's old. That's 1.2a, and the latest version was 2.7, at least 5 revisions of difference (based off of the loader files that if2a had available)

I'd recommend getting the latest version of PowerWriter (v2.61), which will automatically load the latest 2.7 loader by default, so you should just leave the box blank in the Options dialog box. I'm not sure if this will fix the issue, but it will probably help a bit. Also, try the Select+A thing again while linking up with the latest PW version.

Is "GBAldr" the same thing as "GBA Loader 3.3"? I've tried that a couple of times for fun and found that it does work on the F2AU though I think it was originally made for the regular F2A. #127515 - dantheman - Thu May 03, 2007 1:23 am

Yeah, that's the first thing that came to my mind as well, but he's talking about a GBA loader app for a flash cart that lets you store more than one homebrew title at a time on a cart. For F2AU users, this menu is called CIZ X.X, where the Xs are replaced by a version number, but apparently this "GBA Loader 3.3" (named loader33.gba when I found it) will also load GBA binaries correctly on a F2A Ultra cart.

EDIT: Looking at the names that the if2a guys gave these loaders, apparently "GBA Loader 3.3" was originally made for F2A Pro users, which is why I'm slightly surprised that it boots files on a F2AU as well. I'd imagine the save management doesn't work, but still.

In any case, Tones001, another option you could try would be a complete reformat using FlashManager by Mootan. I think the latest beta versions available in English will support the F2A USB cable and F2AU carts - I've used it before, and it works, though I prefer PowerWriter for most functions.
