#12383 - tepples - Sun Nov 09, 2003 10:16 pm

Goomba plays Game Boy games (such as Pokemon Red, Blue, Yellow) and Game Boy/GBC dual mode games (such as Pokemon Gold, Silver) but does not play Game Boy Color only games (such as Pokemon Crystal). It's unlikely that GBC only games will be emulated in the near future because they run the CPU twice as fast, and the GBA's CPU can't necessarily keep up.
