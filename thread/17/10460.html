#93865 - MrD - Fri Jul 21, 2006 7:05 pm

http://forum.gbadev.org/viewtopic.php?p=90101#90101

If the GBA homebrew boots in an emulator, but doesn't work on hardware, check to see if the Nintendo banner is displayed on the 'GAME BOY' "whoosh" screen. If it isn't displayed, the Nintendo header may be missing from the .gba file.

You say nothing happens when you don't have the Superkey inserted... that's strange. 'Nothing' happening is very strange behavior for a DS with no alterations done to it. :) All the Superkey would do is allow you to use the Supercard in NDS mode. If you use the DS menu, and select 'Play GBA Game', you can use the Supercard in GBA mode. You can launch GBA games from the Supercard in either mode.
