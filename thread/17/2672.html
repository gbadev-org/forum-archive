#15022 - Mill - Fri Jan 16, 2004 5:09 am

Hello,

I was toying with F2A Ultra 1G for a while and I decided that it's time to share my experience.

First, F2A Ultra claims that it has 8M SRAM but according to the user's guide, you can use only 1M space for this purpose. Other spaces are reserved for the realtime save (aka snapshot, diehard mode). Now this 1M space (half of other flash carts) should be enough to hold TWO 512k save games. But to support 1M flash save of some recent games, F2A Ultra seems to be using the whole 1M space for just one savegame. So basically you can hold only one save game. Generally, I use the flash cart only to play the games I purchased and if I can carry only one game, this cart is meaningless because I can just carry my original cart.

Second, F2A Ultra claims that it handles all the save types but I couldn't save the game when I tried "Sword of Mana" and "SD Gundam G Generation Advance" (Rom images were dumped from my original carts).

Third, the buttons on the cartridge are very very annoying especially if you, like me, don't care about the realtime save feature.

Anyway, as someone else pointed out, I had good experience with F2A non ultra. That's the reason why I bought F2A Ultra immediately. But so far it falls far behind my expectation.

So, until they resolve these save issues I wouldn't recommend it to anyone who wants to play the game with save feature.

By the way, if anyone is interested, my current favorite is EZF Advance. I just wish that EZF Advance 1G will be available soon...

If someone else has better experience or knows any alternative save solution, please let me know.

Thanks,

#15230 - halljames - Mon Jan 19, 2004 12:35 pm

I have seen numerous posts across the boards about people not understanding how the save system works on F2A Ultra Carts. The 1M save area is designed to hold the save info for the current game being played. When you choose a different game, the 1M save area is compressed to a file and stored in the rest of the Flash RAM. When you next come to play the game, the appropriate file is uncompressed and saved into the 1M save area so you can resume your game.

This works just like pogoshell.
