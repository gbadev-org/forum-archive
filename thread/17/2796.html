#15892 - tepples - Tue Feb 03, 2004 4:53 am

Flash carts themselves use SRAM. If you don't tell the flash software what save type a game uses, the flash software should assume it's SRAM for simplicity.

To convert games with other save types (serial EEPROM, 8-bit flash) to use SRAM, there are "save patches" for many games if you know where to look.
