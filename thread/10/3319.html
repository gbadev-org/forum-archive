QUICKSTART

- Add any midi files you need to the included mid directory.
- Run tr(ENGLISH).exe (this creates the song.gba)
- Flash the song.gba file with your flash-software of choise.


USING

Directional Pad - SELECT MIDI FILE
A = PALY MIDI
B = STOP

START = OPTION
L= PAGE UP
R =PAGE DOWN
