#81779 - SeanMon - Tue May 02, 2006 1:51 am

Download the devkitARM kit for Mac OSX at sourceforge if you haven't yet, double-click the downloaded installer if it doesn't open by itself, and authenticate and install.

Then in Terminal run the following code:
Code:
echo "export DEVKITPRO=/opt/local/devkitpro
export DEVKITARM=/opt/local/devkitpro/devkitARM
export PATH=$PATH:/usr/local/bin:/opt/local/devkitpro/devkitARM/bin" >> ~/.profile
Then type in:
Code:
export DEVKITPRO=/opt/local/devkitpro
export DEVKITARM=/opt/local/devkitpro/devkitARM
export PATH=$PATH:/usr/local/bin:/opt/local/devkitpro/devkitARM/bin

This sets up the correct enviroment variables to the compiler and tools.

That's it (I hope).
