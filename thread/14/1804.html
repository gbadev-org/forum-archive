I don't get this when i load my work using the boycottAdvance, but the message "Loaded Battery" appears when i load my file using the Visual Boy Advance and disappears after the .gba file loads - does anyone know what this means exactly?

#9205 - tepples - Wed Jul 30, 2003 5:04 pm

If your program writes to battery-backed cart SRAM, VBA saves what it wrote to a file on the disk. "Loaded battery" appears when VBA loads the save file the next time you run the program.
