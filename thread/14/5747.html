Well I do not believe that there really are any "Game Makers" for the GBA. There are several Libraries such as HAM that help in the creation of GBA Games as well as tools such as Dragon Basic (which right now doesn't seem to be available) that hide GBA specifics behind the easy to use BASIC language. Other than that there aren't really any short-cuts to GBA development.
