#170558 - dantheman - Thu Oct 01, 2009 10:51 pm

My guess is that he used NDStation, which confuses people by taking in a .gba file and outputting a .nds file. However, it only works when a 3in1 expansion pack is inserted in slot-2. This is because the .nds file is really just the GBA binary with a small loader program attached to the front. When executed, it loads the GBA file into the 3in1's RAM and then launches it in GBA mode. Without a 3in1, it will not work.
