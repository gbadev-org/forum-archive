#54986 - tepples - Sat Sep 24, 2005 5:02 am

To convert .mid to .wav use Winamp. Then .wav will still be huge, but you might be able to use GSM compressed audio.

To play .mid directly, you'll need a sound font, a mixer, and a Standard MIDI File interpreter.

#54994 - headspin - Sat Sep 24, 2005 9:21 am

You can use ModPlug Tracker to convert the MIDI file to a MOD/S3M/XM etc. then use one of the many available MOD Players such as:

- AFM (Another F**king Module Player)
- Krawall Soundsystem
- AAS (Apex Audio System)
- Fras Player
- MASO System
- Music Wave
- NMod v1.c

For streaming audio use Tepples GSM codec, and another way to convert to WAV is to use ModPlug Tracker's Save As Wave... feature
