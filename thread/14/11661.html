#108521 - thegamefreak0134 - Thu Nov 09, 2006 9:20 pm

First of all, my bejeweled game on the site doesn\'t contain a proper header, so you will need to use the EZloader to load it. This is likely the case with the other games. Also, I have found that with this cart it is often wise to burn only the game, and not use the loader at all.

You shouldn\'t need to use the .gba extention, I believe it supports the .bin extention just fine. (Of course, try that anyway just to be sure.) When you boot just the game, does the nintendo logo show, or is it all garbled? If the logo is garbled, the game has a bad header (common among early homebrew) and you will need to fix it using the gbafix tool which I believe can be found on the site.

If the header (aka, logo) shows fine, and the game still doesn\'t run, you have more serious issues, likely with the cart itself. You might try running the RAM and ROM checks in the EZclient menu.
