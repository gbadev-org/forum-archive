#5443 - Daikath - Sun Apr 27, 2003 10:00 pm

First download devkitadv, go to tools -> compilers. on www.gbadev.org.

Go to www.gbadev.org and go to tools, misc and then gba ap wizard, install it.

Then you should be able to choose a GBA project from that, but make sure that all the directories in the makefile are correct (especially the one to devkitadv gets screwed up). Also another important note, dont concider those headers invalible, always check if the REG_VCOUNT in the header is defined as a volatile pointer. Otherwise stuff wont move on your screen.
