Code:
C:\devkitadv>dir /b /ad
arm-agb-elf
bin
crtls
doc
example
include
lib
share
testsuite
tools

Don't unzip each zip file into a separate folder. Unzip them into the same folder, just as you would if you were installing DJGPP. (DJGPP's djdev corresponds to DevKit Advance's core and newlib.) For example, the gcc and binutils zip files both contain files that go in .../devkitadv/bin.
