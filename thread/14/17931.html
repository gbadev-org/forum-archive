#177831 - elhobbs - Mon Mar 25, 2013 5:00 pm

no, it behaves differently. whether this is intentional or not, I could not say.

libnds
Code:
uint32 keysDown(void) {
//------------------------------------------------------------------------------
return (keys &~ keysold);
}


libgba
Code:
u16 keysDown(void)
//---------------------------------------------------------------------------------
{
u16 tmp = Keys.Down;
Keys.Down = 0;
return tmp;
}
