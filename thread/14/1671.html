#8484 - niltsair - Fri Jul 11, 2003 9:32 pm

1024 in 256colors.
2048 in 16colors.

Notes:
-For every map you use, you lose 32tiles in 256colors, 64 in 16colors.
-A map can't access more than 1024tiles (unless some hardware tricks).
-You can have has many tiles as you'd like (given it fits inside a 8megs cartridge), What's important is how many tiles are loaded at any given moment. You can just load dynamicly the one you need.

#8530 - sgeos - Sun Jul 13, 2003 3:46 am

superx10 wrote:
i hear there are two palettes, one for sprites and one for bg's. the sprite one can be either 256 color or 16 separate 16 color palettes.. does the same apply for the backgrounds?


That depends on the mode you are using. Using scale/rotate bgs you can only use one palette of 256 colors. Regular text bgs can use 256 color or sixteen palettes of 16 colors.

-Brendan
