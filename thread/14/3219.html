That's text that's not written in English. Perhaps it's Greek or Russian or Japanese or something. To get your OS to display the text properly so that people who have worked with different scripts can help you, you will need to install more language support modules. In Windows 9x, you can install them from Windows Update; in Windows 2000, try Start > Settings > Control Panel > Regional Options. What are the official languages where the Meteo maintainer resides? When in doubt, try Japanese first.
