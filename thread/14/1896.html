#9692 - tepples - Thu Aug 14, 2003 3:12 am

If you have four "text mode" backgrounds (that is, mode 0) with unique 256x256 pixel maps and 16-color tiles, there is room for just under 2000 tiles in VRAM, starting at pattern base block 0, 1, or 2. Each "text" mode map can address up to 1024 tiles without vcount tricks. Remember that if you need lots of unique tiles but don't need all of them at once, you can write new tile data over tile data that you're no longer using.
