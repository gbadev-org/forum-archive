#136053 - tepples - Sat Jul 28, 2007 7:58 pm

"Running" means you download compiled programs and run them. "Making" means you write source code files in the C programming language, compile them, and run them.

You will need GBA compatibility mode to run programs from before 2005, as well as programs from 2005 or later that are designed for the Game Boy Advance system. An EZ-Flash 3-in-1 card will let you use GBA compatibility mode with your R4.
