#36130 - sgeos - Tue Feb 15, 2005 6:38 am

I've used MapEd extensively, but was not responsible for creating the art. IIRC, tilesets need to be saved as 256 color BMP images. The program only uses the first 16 colors of the palette. I think if you save as a 16 color BMP, and then resave it as a 256 color BMP, the tile pages will load and display properly.
