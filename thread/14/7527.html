#61330 - chishm - Sat Nov 19, 2005 12:18 am

No, you will need to recompile and in most cases modify the source code to run as a multiboot. This is because non-multiboot games expect a flashcart to be in the GBA, and willtry to load stuff from one even if it isn't there. Any game that doesn't do this has most likely been compiled as multiboot anyway, unless the author was ignorant of the non-flash card users out there.
