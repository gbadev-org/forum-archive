I was having the same problem. In my case (and i'm guessing yours as well), the problem was that I never actually put grit on my computer!

Thats one of the more blatant shortcomings of this tutorial - it never actually tells you that you have to obtain grit before anything will work, leaving you to assume that it is built into the makefiles or something. The more aggravating problems are hidden in the form of obscure code typos provided on the online tutorial. I've found the best approach is to go ahead and READ everything in the tutorial, but when it comes time to type up your code you should just copy it from the already finished code provided in the download, because that code is actually correct.

Anyway, as for your problem

Go to this page -
http://www.coranac.com/projects/grit/

download the latest binary - the link will say:
grit-0.8.3.zip

unzip into your devkitpro directory.

Grit comes with it's own FreeImage.dll file, which is part of the reason that if you tried to download FreeImage.dll on its own you still had problems.
