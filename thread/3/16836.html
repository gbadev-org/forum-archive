...search for "GBA Multiboot" on Google and you'll find a small list of homebrew games you can run. There is no way to run GBA files larger than 256 KB on a GBAMP. It is not possible. If you have a slot-1 passthrough device you can use the GBAMP to launch most DS homebrew, but that functions entirely differently.
