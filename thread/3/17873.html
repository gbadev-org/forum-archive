#177529 - sonny_jim - Wed Aug 01, 2012 6:33 pm

Think about what you are asking. The games were designed for 256x192, which means all the artwork/graphics/etc are all at 256x192 or smaller.

Simply put, "quality comes from the source of the stream"

You could upscale them all you want, but it would still be 256x192 resolution and therefore would look 'blocky'.

You'd need to rewrite each game from scratch if you wanted them to run in a higher resolution, which is damn near impossible and beyond the scope of this forum.
