#332 - f15hy - Mon Jan 06, 2003 4:15 pm

Had the same problem. I think i fixed it.

I'm using winXP home.

In your device manager(control panel-->system), extend "Universal Serial Bus controller."

I'm using USBv2 so it might differ abit.
Go under the properties of "USB 2.0 root hub"
Under power management tab, uncheck "allow the computer to turn off this device to save power".

I did this for the rest of the "USB root hub", unsure which 1 is for which usb slot.
