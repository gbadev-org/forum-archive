#2643 - tepples - Sun Feb 09, 2003 11:57 pm

isophage wrote:
Would it be possible to use a link cable (Such as the Madcatz GBA Gameport -> Parallel Port cable) to transmit the data sent from GBA->GBA over the internet

In theory, it would be possible, but it wouldn't be able to emulate a normal Game Link cable because of the latency; commercial games expect a latency a couple orders of magnitude faster than even a high-speed Internet connection can provide. Games would have to support this type of device separately and use prediction to calculate where the other players are likely to be at any given time.
