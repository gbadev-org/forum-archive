nixendra wrote:
i've also considered a flashcart-sd/cf but do i need to take the GameboyMovie or any would do (different rom bios / loading method ?) ?

GBA Movie Player works for running "multiboot" programs, either stand-alone multiboots or those that access the FAT file system on the CF card. M3 and SuperCard are more expensive but they let you run homebrews that were originally designed to run from traditional GBA flash cards.
