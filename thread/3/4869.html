#40309 - cocole - Sun Apr 17, 2005 9:38 am

If your cart is an EZFA then Borden made a driver for ya.

Quote:
EZFA Keyboard Emulator


--------------------------------------------------------------------------------

EZFA Keyboard Emulator is a software tool to test Keyboard Emulating function between PC and GBA by EZFA Cable. This function will be integrated to the new version of EZFA Client in the near future.

1,Software List
KeyBoardemu.exe KeyBoard Emulator Program
EZFAKemu.gba Keyboard Emulator run on GBA

2,Preparation
GBA Emulator software installed on your PC(such as VisualBoyAdvance)
EZFA Client installed on your PC
EZFA Cable and EZFA Cart

3,Operation Instruction
a,Connect your PC with GBA by EZFA Cable
b,Run EZFA Client
c,Download EZFAKemu.gba to GBA and run it. You could run it in two ways, one way is to download EZFAKemu.gba as a normal GBA game,and anothe way is to download EZFAKemu.gba directly to GBA RAM and run it.You could get download details in EZAF User's Guid.
d,Close EZFA Client
e,Run KeyBoardEmu.exe
f,Choose a key for each button on GBA,and set keyboard map by clicking KeyMap
g,Click Hook button to make above keyboard-map setting effect.Stop button to disable keyboard map
h,Open GBA Emulator,and set the corresponding Joypad map as the keyboard map just set on EZFA Keyboard Emulator
k,Play your game in GBA Emulator
