A binary patch, such as an Action Replay code (Action Replay products were sold as GameShark brand in some markets), works only on one build of a program. The versions of a game released in North America, Japan, and Europe are almost definitely different builds. Sometimes, a game publisher will fix bugs in a version 1.01 of a game, which is another build, and it throws off the binary patch as well.

I'm guessing either 1. you have a different build of the GBA game than Datel used when making the code, or 2. you're trying to use a code for Super NES Super Ghouls and Ghosts with a GBA version.
