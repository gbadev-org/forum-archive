#12321 - tepples - Sat Nov 08, 2003 12:14 am

The F2A linker has a USB device controller and a GBA compatible host controller. I don't know whether they're on the same die.

I'm guessing that like the MBV2 cable, the parallel port F2A linker, has only a GBA compatible host controller; a parallel port device controller is trivial enough to include on the same PIC as a GBA compatible host controller.

If you want a "dumb" cable, where the PC acts as the GBA compatible host controller by manipulating individual wires of the parallel port, try the XBOO cable.
