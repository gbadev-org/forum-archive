#125498 - dantheman - Fri Apr 13, 2007 11:22 pm

The R4 is a slot-1 card. You can get a slot-2 card that fits in the GBA slot if you want. Both the same basic thing, and you don't need to have both in order to run homebrew equipment. If you get only a slot-2 card, you DO need a slot-1 NoPass device, but these are usually cheaper than actual slot-1 homebrew devices like the R4.

If GBA support and extra RAM for DSLinux use does not matter to you, then you will not need to get a slot-2 device at all after getting your R4.
