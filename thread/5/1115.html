Darrell Blake has released a source code to display pictures in a slideshow:

So far the program accepts 10 images called image1.h, image2.h... image10.h. Use
whichever program you wish to create the header files (or stick them all in one
header file if you wish) and then run make.bat. Once you have your glorious
rom run it in your emulator or GBA and use the A button to go to the
next picture and the B button to go to the previous picture.

This C source is very simple, so it is ideal for beginners.
