A new version of RipaRoo, the Arcade Games Editor
has been released.

Version info:
v1.0 - Removed all need for any external tool to figure out grid size.
- Added in app realtime width and height adjustments.
- Added JASC palette support.
- Added splitter window for both a view of source and dest bmps.
- Removed buffer file requirement.
