News > Catapult demo competition

#1406 - SimonB - Sat Jan 18, 2003 1:13 pm

There will be a competition held to see who can develop the best demo using
the V1.20b version of Catapult - when it is released. See
http://nocturnal.proboards4.com/ for more details.

There will be a cash prize for the winner and runner-up.

V1.20b will be released around the end of the month, and will include a host
of new features including :
- full sprite/background rotation/scaling
- multiple script support
- keyword coloured IDE
- up to date ROM's for all the demo projects
- enhanced image & tilemap editing tools

Additionally, Nocturnal is pleased to announce they have licensed NEiM0D's
NMod player for use in Catapult. It is expected V1.30b will be the first
version to fully support NMod.

More info about Nocturnal can be found on www.Nocturnal-Central.com
