Version 1.0 of AGB Rogue has been released by Donnie Russell II:

I recently updated my GameBoy Advance port of Rogue, AGB_Rogue, to version
1.0. The most notable changes include:

* Larger text for the title screen and key selector popup
* Automatic closing of key selector popup
* More optimized graphics code and brighter colors
* Character representing gold is now an asterisk
* Scrolling of levels occurs in three sections instead of two
* Use of the START and SELECT buttons for moving diagonally and running
* While holding down START or SELECT, the selected action is displayed,
and can be reset to default or cancelled

AGB_Rogue and other Roguelike games for various systems can be downloaded
from my home page:

http://www.freewebs.com/drussell/

You can download AGB Rogue 1.0 straight from the main site!
