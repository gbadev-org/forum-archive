Mines for the gameboy advance has been released by Vortex:

This is my first GBA game. Tested using VBA and BATGBA.
Should work on hardware.
The source code is available upon request.
Game Controls:
D-PAD: Move cursor in playing area
A: Reveal Mine
B: Mark Mine
RIGHT: Automatically reveal all surrounding cases not marked by a flag
START: New Game

Download the game straight from the main site.
