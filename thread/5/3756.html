A nice console input demo with source has been released by Hu YingHao (Eric):

This demo is an input program for gba. You can use it to select and input some chars. This IME program is created with csagvlib(http://www.loveu.cn/hdxc/fenye/youxi.htm). It is only a concept model for gba development. This program shows that gba can be used for some simple text programs such as console program. It will greatly extend capability of the gba program.

You can get the input demo and the source code in C straight from the main site!
