Tweakmode 3d demo:

Hey there, MrMr[iCE] again with a cool little demo I whipped up over the weekend.
The rom shows off a few things, like mode splits on the screen, fast 3d vectors, and
a tweaked mode 4. Similar to tweak mode 5, but designed to work with mode 4,
providing room to flip buffers in a single page of vram. Check the readme, source
is included. Mirror at http://www.bigredpimp.com/MrMister/tweakmode.zip

You can also get it from the main web site.
