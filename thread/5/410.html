#2590 - SmileyDude - Sat Feb 08, 2003 4:01 am

ok, but if you are porting a game that is in MAME, then you have all the info you need to just convert the data from the format it's stored in the ROM -- just write some code to convert the data on either the PC or the target machine (for example, the GBA). I used this method with my partial BombJack conversion, and it worked fairly well.

Also, by doing it this way instead of taking screenshots, etc, etc, the layout of the characters and sprites in VRAM is pretty much identical to the way it is on the real game -- which means that it's pretty easy in my case to drop a Z-80 core into it, and make it into an emulator (I know this because that's what I did with it :) )
