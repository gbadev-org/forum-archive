Jeff has just sent in a few big news items related to Dragon BASIC:

The SimForth site is gone, and Dragon BASIC has switched hosting
services. The new website is http://www.jm-basic.com/ I encourage
everyone to update their links and stop by the new site and let me
know what they think.

Next, Dragon BASIC 1.4 is about to be released in the next few weeks.
It is a major leap forward not only for itself, but compiler design in
general. It gives access to the underlying language: TIN and the
entire Dragon BASIC API is open. You can create your own API functions
in TIN or ARM assembly and PROTOTYPE them for use in Dragon BASIC
code. Dragon BASIC is now strong-typed and can handle multi dimensional
arrays and much more...

While working on the new site, the download and registrations will be
inactive until 1.4 is released. For those interested, after 1.4 is
released, I'm going to be working on an e-book for programming the GBA
using Dragon BASIC.
