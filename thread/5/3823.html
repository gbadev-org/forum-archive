yaustar has submitted a great third year university project based around Metal Gear by Konami:

This is the final version of a third year project for University
between myself (Steven Yau) and MGFox (Andrew Buckingham). The game
design is copyrighted by Konami but everything else is written from
scratch and there is a supporting document to read in PDF as well. Have fun!

Bear in mind that we knew very, *very* little about the GBA hardware....
and we still do :p

Here is a link to the project page to find out more: http://yauster.com/dev/gba_mgs2d.html

The included documentation contains all of the steps taken to develop the project
and is a great resource for anyone that is thinking of making this style of game.

You can download Metal Gear straight from the main site!
