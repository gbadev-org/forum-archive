#12990 - krom - Wed Dec 03, 2003 12:57 am

A GBA text editor has been released by Francesco Napolitano:

Since I discovered that a project named Keyboard Advance yet existed
(it's about connecting a real keyboard to the GBA), I finally decided
to rename my Keyboard Advance (the GBA text editor) into DanTE Advance.
The new release includes some minor corrections and bug fixes and can
be found on www.danteadvance.cjb.net. The old site is no more active.

To find out more about the text editor, check out the homepage at: http://www.danteadvance.cjb.net
