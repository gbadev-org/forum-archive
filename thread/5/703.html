News > New Sound Tool: TektronicWave

#3630 - krom - Sun Mar 02, 2003 7:35 pm

Lucian Darie has released new tool for GBA sound development Tektronic Wave:

This tool is a wave to C/C++ converter/generator mainly designed to use
in GBA sound development.The software is able to convert multiple input
to output wave formats.
Based on a practical Windows graphical user interface, it is possible to
form a list of input waves, set the output desired format
(8-44.1 KHz sampling rate, 8 bit, Mono for GBA) for each wave file in list,
test the converted format and finally to generate C/C++ module and header files
suitable to use in GBA sound development. Most of the PCM formats are supported
for input waves files (8/16 bit, Mono/Stereo/8-44.1 KHz).

See more info at: http://www.tektronic.home.ro

Download the Sound Tool straight from the main page.
