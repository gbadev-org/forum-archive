#27605 - caitsith2 - Sat Oct 16, 2004 8:40 pm

I just finished ripping the music from your game, to the upcoming GSF format. 3 music tracks, right?

It took me a few hours to track down the music process function, and the music selection functions. Then I added some code to cause the code handler to go into a dummy vblank routine, and make the music process code wait for a vblank interrupt with the swi interrupt, rather than by the if VCOUNT == 0 loop. Doing it this way, saves battery power on real hardware, and real cpu cycles on an emulator.
