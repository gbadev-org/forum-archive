Damian Yerrick has released a nice game called Vitamins:

Vitamins is a falling pills game. The object is to get rid of the space
invaders by drugging them up. Manipulate the falling capsules to
make four in a row of one color.
This "milestone 1" release is lacking major "polish" features, but it's
completely playable (and only 18 KB in size). Let it go on record
as the first Dr. Mario port on the GBA platform.

It has been tested in VisualBoyAdvance and on hardware.
It includes full source code (compiles with Devkit Advance;
requires GBFS tools).

You can download it straight from the main page!
