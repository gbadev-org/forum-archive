A great puzzle game concept has been released by Nrx called Mario Balls:

I've recently decided to have a look to the GBA; my first little achievement is this "Mario Balls".
You'll find screenshots & a readme file in the zip archive attached, together with the ROM.

For future updates, check the Player Advance forum (in French):
http://www.playeradvance.org/index.php?showtopic=6714

Note: The source code is also available (check the link on the PA forum);
the game is to be recompiled using VisualHAM (there is a "HowToCompile.txt" shipped with the sources).

The demo features great use of rotational screen effects and is very playable.

Download the game straight from the main web site!
