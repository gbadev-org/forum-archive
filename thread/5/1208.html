#5989 - krom - Wed May 14, 2003 8:32 pm

Sammy Fatnassi has made a GBA slide-show creator called SlideAdvance:

It allows you to easily create a SlideShow that runs on the GBA, no
Programming/Compiling needed.
For each picture, you can add Text and select a Transition Effect.

Features:
- Import all basic picture's formats
- Allow you to write Text for each picture that can be displayed over them.
- Allow you to pick a transition effect between each picture.
- Create Fullsized pictures that can be scrolled.
- Create Resized pictures that fit inside the screen.
- Create Thumbnails that can be browsed.

It has been stored in the GFX Tools section of the site.
