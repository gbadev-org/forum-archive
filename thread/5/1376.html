A new version of Mines Advanced has been released by Vortex. There are a lot of changes
and improvements, including:

* The source code is included (GPL license)
* Basic sound support
* Larger game field
* DMA ClearScreen for Mode 3 implement
* Cleanup the main loop
* Fade In/Fade Out effects
* Animated flags

Mines Advanced v1.2 and it's C source can be downloaded from the main site!
