News > TileMax 1.6 has been released

#5828 - SimonB - Fri May 09, 2003 2:17 pm

Sean Reid has released a new version of TileMax. From seanreid.ca:

TileMax 1.6 has been released. Finally, TileMax is no longer a thread-beast. I fixed up a major bug whereby threads where not being correctly cleaned up. Also, I speed up other areas of the program to make it more efficient.

New in 1.6:
- Removed the run-away creation of threads, providing a major speed increase.
- The map editor now paints tiles with the correct palette on the first load.
- Prevented tile-view redrawing when changing selection size.
- Added better support for 16/16 colour palettes.

For more info go here: http://seanreid.ca/project/TileMax/
