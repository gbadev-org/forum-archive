and from the readme...
Version History
===============

1.5, 03.09.03
- Changed the palette editor such that the marker is the palette index number
instead of a black dot.
- Allowed negative increments in the colour-editor auto-filler.
- Added support for 16 16-colour palettes.
- Disabled constantly-executing code that happend from holding shift and not
moving the mouse.
- Correctly disabled threads of inactive tile set and map editors, thus
increasing speed greatly.
- The map editor now draws stamps based on a pre-rendered image instead of
rendering tiles repeatedly.
