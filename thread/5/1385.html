Cutter V1.1 has been released, Cutter is a tool to help you create your
tiles or sprites tables, preserve inside antialiasing of prerendered sprites.
It also has some great optimisations functionnalities.
New in V1.1 :
- Improved GUI ( tool bar and more friendly overall user experience )
- Keyboard shortcuts.

http://www.phelios.com/dev/
for a trial download.

system requirements:
Windows 2000 XP or higher ( some functions are not available under 95/98/ME )
Quicktime 5 or higher ( pro not required )
