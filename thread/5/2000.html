News > WAV file plug-in for Pogoshell

#10346 - krom - Wed Sep 03, 2003 12:12 am

A new plugin for the Pogoshell OS has been announced by Pat Crowe:

For the sake of completeness rather than a technical
breakthrough - I have released a WAV file plug-in for
pogoshell.

Use it for pretending that your GBA has an MP3-like
music playback capability. Of course, the size of the
files is the problem.

It plays back WAV files in pogoshell. The files
currently have to be 1 or 2 channel (mono or stereo).
They can be any number of bits per sample up to 16.
they can be at any bit rate.

To reduce the size of your file to a sensible level
use an application such as Goldwave to convert
your wav file to 8 bit mono, with a sampling rate of
18157 bits per second. This gives a smaller file size
optimised for the playback sampling rate.

Get the (free) download at www.bookreader.co.uk
