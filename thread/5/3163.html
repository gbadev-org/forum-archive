#18740 - krom - Fri Apr 02, 2004 8:16 pm

A new image converter in the works has been released by Lupin:

This is my 24 bit bmp to 8 bit binary image converter, it will take
all kinds of bitmap images and throw them together to one dithered 8 bit
image with one color optimized palette. At the moment it can only export
8x8 tiles and therefor only be used for sprites and map data, but more
features will be added later.

It has been stored in the Tools/GFX Tools section of the main site!
