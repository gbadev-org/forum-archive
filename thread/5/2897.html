Donnie Russell has released AGB Hack version 1.0:

My second GBA game is now available from my web site:

http://www.freewebs.com/drussell/

(On the Download page, the third entry in the RevivedHack section)

AGB_Hack is a highly accurate adaptation of PC Hack 3.61, released in 1986.
Hack started out as a clone of the original Rogue, but was later expanded
and eventually became the basis for the modern classic NetHack in 1987.

The included guide and readme files are absolutely essential reading before
starting a game, even if you have played Hack or NetHack in the past.

You can download the game from the main site!
