MumblyJoe has released a new tool called Columniser

This program will take a series of bitmap images and convert them into a header file with a data and a palette array that can be included in games. The advantage of this is all your sprites share a palette and it has a Windows interface, not command line, and also has a couple of options for array names and transparency.
