A new GFX Tool has been written by Alexander Gebers called GBAMapEdit:

Integrated:
* Loading an Tile-Picture
* Edit Maps with 4 Layers and max. 1000 Tiles
* Creating of Tile-Brushes
* save Map-Data as C-Source Header File
* Load/Save Map-Data in my own format
* Tilesize: 8x8 Pixels
* Max. Map-Width: 100 Tiles
* Max. Map-Height: 100 Tiles

You can download it from the Tools section of the main site.
