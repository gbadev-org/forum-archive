Nocturnal has released the latest version of Catapult - V1.21b.

Catapult is a development system for the GBA - new features for this version
include :

- all demo restrictions have been removed (fully featured).
- new Copter game, with complete source.
- in depth tutorials (the first six).
- revamped GBA engine - interpreted scripts replaced by compiled scripts.
The new ROMs run 20x faster.
- new 'Export' system and pricing model.
- the latest ROM samples (including 4 playable games).
- new language commands and editor features.
- lots of bug fixes.
