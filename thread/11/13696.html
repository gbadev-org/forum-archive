Dicewars DS is a light-weight online strategy game in a Risk-like fashion. If you haven't heard of Dicewars before, check out the excellent original flash game.

I'm now almost ready with the v0.4.0, which is also going to be the next public release. However, before that the game needs throughout multiplayer testing. If you're interested in helping out, here's what you need:

o Nintendo DS handheld console and means to run homebrew on it
o Ready configured and working wifi for the DS
o Some/severals hours of free time to join the game on Thursday 20th of September. Gaming and testing will happen after 18 GMT.
o Always up to date information about testing and downloads can be found at http://dicewars.drunkencoders.com/beta/

Each game can host max. 8 players and I'd like to see how the server acts with multiple games going on at once - so invite your friend as well!

If you want to join the game, just join blitzed irc on channel #dicewars on Thursday. I'll be online most of the time also on #dsdev and #dswifi.
