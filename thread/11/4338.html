#29143 - bertsnks - Fri Nov 12, 2004 3:40 am

So kind of like a real life alarm clock?

Unfortunately after the GBA is in sleep mode, it can only be woken up by either an Key, Cartridge or SIO interrupt. Needless to say, all GBA devices such as sound, LCD, etc are stopped. The CPU is semi-stopped. The only thing the CPU then does it wait for the "wake up" signal and therefore no clock wake up code can be run at the same time.
