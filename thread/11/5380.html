Nintendo DS/GBA programmers - AIS

Job Description
This well respected, award winning independent studio is currently looking to add two experienced programmers that can bring technological innovation to games developed on the Nintendo Game boy Advance and Nintendo DS. He/She will be responsible for maintaining code quality on GBA/DS projects while being a core member of the product leadership team.
Will be responsible for the programming schedule and all technical decisions and helping develop key technologies and software architecture for GBA/DS games. Areas of involvement include 2D/3D graphics engines, sprite animation systems, audio engines, script systems for game logic and many other game related technologies, and participating in game design with the rest of the team. Define and track programming schedule while working with designers on product and technical designs. Must have good communication skills and the ability to work collaboratively with designers, artists, and programmers, and have a strong passion for gaming.

Requirements:
Candidate should have a BS in CS/MATH/Physics.
5+ years of Extensive C, C++ programming experience
Strong mathematics, real-time, high performance code practices and optimization techniques
Program low-level elements with Assembly language
Excellent debugging and trouble-shooting skills
Familiar with professional programming techniques
Work with team members to meet project schedule deadlines
Commercial Game development experience preferred

Responsibilities:
Participation in program conception and architecture
Coding, maintenance and debugging all game elements
Design, specification and coding utilities necessary to the development of the game
Reasonable documentation of the game code or associated utilities
Maintenance and archival of backups of various versions of source code, art and other data used in developmental process
Interact with other programmers, artists, designers, producers and others as necessary during game development

Knowledge and experience:
Experience programming 3D game engines
GBA or DS experience not required, but the GBA or DS platform should be candidate's #1 focus/interest.
C++ and OOP design experience required
Experience writing efficient C++ and Assembly code
Should be prepared to provide code samples

Skill Required:
Proficient in C/C++
GBA/DS game development experience (home made demos welcome)
Good optimization abilities, understanding of assembly a plus.
Strong debugging skills
Passion and strong interest focused on pocket and mobile games.
Ability to work well in a fast pace dynamic team environment.
Self motivated and able to work well in small teams as well as solo on various project parts
good communication skills and attention to detail a must.

Candidates must be able to move to Canada area, and start immediately.
Salary: Dependent on experience
Stock Options
Bonuses
Free Medical
Full time position
