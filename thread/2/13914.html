OffTopic > Oggenc2.exe crashing, possible fix

#137532 - DiscoStew - Mon Aug 13, 2007 1:23 am

I wasn't sure where to post this, but the Off Topic forums seemed general enough. Not sure anyone else has had any problems with Moonshell's DPG encoder, or wouldn't if you use a different DPG encoder, but I think having this info helps.

After using the DPG encoder supplied in the newer Moonshell (1.71?), I found that some of the videos I wanted to convert failed during the OGG conversion, making it crash. I searched a little bit on the converter, and found a website where newer version of the encoder could be downloaded, which was v2.83 from v2.82 supplied with Moonshell. I've only tried a few files, but they converted fine unlike the previous version, and they play fine in Moonshell.

The website, although Japanese, has enough English on there to get you through it.

http://homepage3.nifty.com/blacksword/

Download the oggenc2.83 file of your processor type, back up the old file, and extract the one you downloaded. Hope this helps anyone that was having trouble before.
