OffTopic > Important DST setting on Windows XP

#121289 - zzo38computer - Sun Mar 11, 2007 7:39 am

See the TZI value for your time-zone at http://support.microsoft.com/kb/928388 and then navigate to "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Time Zones" and enter the new TZI value (if you want, rename TZI to TZI_old_value and create a new TZI for the new time-zone settings, in case it breaks you can restore the old one again.) And then go to system time setting, select the time-zone, apply, and OK to copy the setting to the current control set.
