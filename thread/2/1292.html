#6481 - Daedro - Tue May 27, 2003 2:01 am

Okay it worked.

--------------------------------------------------------------------------------

Go to "Start" on the task bar.

Click on "Run.." button.

Type: regedit

In regedit go threw the folder tabs like so:

HKEY_LOCAL_MACHINE
\SOFTWARE
\Microsoft
\Internet Explorer

Then Right click on "Internet Explorer" and choose New --> Key.

Name it to "View Source Editor"

Then in the "View Source Editor" you make a folder called: Editor Name

On the first and only file in that new folder double click on it, called (Default)

In the blank box type in quotations the file name (eg. "G:\WINNT\WORDPAD.EXE")

--------------------------------------------------------------------------------


Hope this helps, also I do not take any responsibility for the affects or risks you are taking in doing this. Regedit is not something to be edited without caution.
