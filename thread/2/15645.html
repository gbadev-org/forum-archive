#158599 - dantheman - Sat Jun 14, 2008 5:09 am

Dwedit wrote:
I do use Dosbox to get Tile Layer running, it's just a royal pain to run Dosbox, type in the Mount commands, then start Tile Layer.

If you drag and drop random.exe onto the DOSbox.exe file (or a shortcut to it) DOSbox will launch, mount the C:\ drive to the folder random.exe is in, and start random.exe. Of course if you need to supply arguments this won't work as well, but for my purposes it's easy enough to pin a shortcut to DOSbox on my Start menu and drag and drop that way.

Or you could use the autoexec thing as ninjalj described.
