Any good reason, other than you're familiar with other products from the same company? It's like buying an Xbox because you like Windows. You can buy a fully homebrew-enabled DS cheaper than a PSP.

Quote:
The problem is, I suspect anyway, if I buy one now, won't it be loaded with 2+ firmware, which if I'm right is imcompatible with psp homebrew?

PSP firmware 2.01 through 2.60 is incompatible with homebrew if a vulnerable version of Grand Theft Auto: Liberty City Stories is not available. This could happen for any of three reasons:
  • The user is a minor.
  • The user lives in a country where GTA is banned from sale or import.
  • The disc is a newer version, pressed after Rockstar found and fixed the buffer overflow.
