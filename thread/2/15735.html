#159332 - silent_code - Sun Jun 29, 2008 10:57 am

Actually, I (personally) don't see it as an RPG... It's a very lightweight RPG at best, but it's definitely hardcore H'n'S (Hack And Slay). Also I don's see any freedom in the game at all...

BUT: The characters' skill trees, the massive hordes of monsters and the core gameplay element: the loot, make up for any lack of story, quests, real NPCs etc! Really! If you like WoW, try Diablo. (After all, it should be called "Diablo: World of Warcraft") ;^)

It has something very special to it - and a very dense atmosphere. It's like PacMan: simple, but entertaining, times over and over. And if you like sporting competition, the online ladder is for you. If you like to spend an afternoon slaying monsters with your friends, the multiplayer is perfect.

And I'm, talking about both games. The original had something, the sequell didn't: Virtually no skills, but all characters could learn all spells. Epic!

I must admit, that I am a little bit excited about the new game (although it's possible it looks like a lot more in this thread ;^) ) - something that happend the last time, they've announced StarCraft2 ... Damn, it's so sweet, my teeths start to hurt! X^D
