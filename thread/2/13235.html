The number of polygons that can be displayed on the screen is dependant on what is happening in the game at the time. There is no definitive answer for any machine, for example the 66Million polygon per second count that was given for the PS2 was a complete lie since no game ever got past [around] 12M. The Gamecube gave realistic in game polygon counts with effects on.
