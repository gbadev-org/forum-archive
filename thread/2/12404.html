#117967 - tepples - Thu Feb 08, 2007 11:03 pm

First you'll need a GBA flash cart with a real-time clock so that the GBA knows what time it is. M3 and Flash2Advance Ultra appear to have RTC; SuperCard and F2A classic appear not to.

Second, you'll need a lot of battery power in order to keep the CPU going so that it can periodically check the RTC. GBATEK states that timers cannot wake the GBA from sleep mode; only the serial port, the gamepad, and the Game Pak IRQ can. The only other low-power state on the GBA is the one that IntrWait() uses. It stops the CPU, but the LCD is still turned on.
