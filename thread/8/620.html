#3255 - peebrain - Fri Feb 21, 2003 6:10 am

Try...

Code:

int myfunction(int whatever)
{
    asm("@ inside myfunction()");
    asm("@ that = whatever + 4");
    int that = whatever + 4;
    asm("@ return that");
    return that;
}


Basically, just put asm("@ your comment here");.
