I've released my first 'serious' DS homebrew project, a remake of the old Spectrum/Amiga/ST puzzle game Deflektor. You can find more details and a download in this post on my site.

You need to DLDI patch before running and place the data folder in the root of your flash card, see the accompanying readme for details. Any comments/suggestions/useful bug reports are welcome. Also I'd like to know if it works on a DSi, since I only have an original DS to test on.
