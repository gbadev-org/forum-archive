DS homebrew announcements > SpriteEditor (SEDS) 0.4 and Animation Editor (AnimEDS) 0.2

#176243 - PypeBros - Mon May 30, 2011 9:48 am

Pixel art fans (and parents), rejoice!

Here's a long-delayed update of my sprite/tiles edition tool for the Nintendo DS, joint with the release of a still-experimental modular animation editor. You can find some snapshots of SEDS and AnimEDS here. Both are open-source projects that compile with libnds-1.4.8 and hosted on sourceforge.

For those who had previous experiments with SEDS, release 0.4 brings in:
* improved Palette editor, now more comfortable to use,
* new functions on the grid: swapping, colour scan, overlays stamping
* bucket-fill tool (L+B)
* many small (obscure) tools to assist colour reduction tasks.
* compatibility with my brand-new Animation Editor for DS.

Art created with SEDS is stored as a DS videoram dump and can easily be reused in nintendo DS games or converted into png for sharing (see user guide for info).

Direct download links: SEDS0.4 and AnimEDS0.2.

The user guide becomes too long to be joint in such posts, so just check it online or press both L and R in the sprite editor for hints.

Both are still in development, so I'm eager to hear any tester's tale. I can provide a skype ID for chatting about the tools on PM request.
