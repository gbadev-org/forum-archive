Thank you very much!

For the top screen I'm working on making it in graphical mode, with the game information: panel size, number of mines, remain flags to put, and a simple miniboard for quick view of what part of the panel is active and which cells remain to discover.

Also I'm planning for the next release to add one more panel size (extra large: 32x32) and a "standard" panel, that is the expert windows minesweeper panel (30x16, 99mines) for those who want to try to beat their windows record with the DS. But sorry no score saves for the moment.
