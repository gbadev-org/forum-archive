#131745 - dantheman - Tue Jun 19, 2007 6:03 pm

jackman wrote:
* select color shades for monochrome GB games like on real GBC/GBA/SGB


I had previously mentioned this idea for Goomba Color, thinking that the color information was stored in each game and could therefore be read by the emulator, but as it turns out, such coloring information is stored directly on the GBC BIOS, with each game having its own entry. As no one has dumped the GBC BIOS as far as I am aware, there would be no easy way to do this other than making our own database of coloring info for each individual game.

As the standard Super Gameboy palettes are fixed, those could be implemented in theory. The updated Goomba Color WIP build currently does only the first SGB palette, and I don't think this includes games with special SGB palettes, but I could be wrong.

Dwedit was the one who answered my question in the first place, so he would know more about this than I.
