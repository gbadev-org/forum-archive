You can Download JXW here

This one's for me!!
If other folk enjoy it, then so be it, but I made this for myself because I was getting really ticked off by the lack of decent crossword games* on the DS.

There's 3479 crosswords in the game.

Each one's selectable from the very uninteresting title screen, and playable whenever you want.

Pick a level, play it, done.

If you want to save midway, hit start to return to the menu, then quit. When you come back, it should be right where you left it.
Only one save is available, though. It only saves the last puzzle you were playing.

Other than that, it's nice and simple.
Reds show up where you've got a letter wrong, and nothing fancy when you're right.

There's no "COMPLETE!!!!!" screen.
If you've got it all right, you should be able to tell... it's a crossword!!

So, enjoy it for what it is.


* Not that the crossword games on the DS don't have good crosswords.. They do.. It's just that stupid UI that they all use is really really really rubbish..

Uwe linked me to this page, where they hold some data files from an old Freeware German Crossword game, ?Kreuzwortr?tsel 2005"

I'm about 98% sure I managed to decypher it ok, and have recompiled a bunch of puzzles, using those German Clues and Answers.

This file contains the new .jxw file.
Pop it into the jxw/ folder, and rename it to crosswords.jxw, so that the game loads that instead of the English one.
Then play the game, and it'll be in German for you.

Note : Although I made sure that all the answers are Plain Letters (ie, work with the keyboard!), there might be instances where the clues end up in bizarre gobbledygook, because they might use odd characters or something. It should still display ok, but I didn't fancy going through 200+ puzzles to check!!


Homebrew sites that decide to "NEWS!" this, should do all the hard work, and then have a different download, specifically ready for folk, in German.
If I add any more languages, I'll probably end up having a different frontend, with selectable languages, or something..

If anyone wants any more languages, point me to some files, and I'll do the whole decyphering/recompiling thing for you.
