#116782 - Metaluna - Mon Jan 29, 2007 3:31 pm

Penko DS is a new MSX 2 emulator programmed by Kyuran, the author of Croco DS.

0.1 alpha release features :

- A resize mode to manage the various resolutions of the MSX.
- DLDI support
- rom and dsk supports
- sound (with same issues than Croco Ds)

Copy rom in the root of your card on in roms/msx directories, use DLDI to patch the .nds.

It's a very early release, so I guess it's OK if it doesn't work on all hardwares yet! I didn't manage to make it work on my M3 DS. I read there are issues with M3 Simply and G6 too. But people with supercard SD should be able to use it.

Don't hesitate to leave your comments on the author's website : http://deadketchup.kyuran.be/

File : http://media.kyuran.be/temp/penkods-dldi-v01a.nds

Anyways, keep an eye on this one. It took only a couple of weeks for Kyuran to make the best Amstrad CPC emulator available and I'm quite convinced this emu will be the next gem in MSX emulation.

PenkoDS v0.2a has just been released.

Quick translation of the new features :

- speed improvements. Many games now have 100% speed.
- mode 2 transparencies corrected
- files shouldn't appear twice in the gamelist anymore
- DLDI support and sound should be better

DLDI support is indeed much better. I can now launch PenkoDS with my M3 SD and it's just... great!

I've just tried a bunch of games : Vampire Killer (aka the very first Castlevania), Pacman and Bombjack... it's really amazing!
