STYXDS is Atari ST emulator programmed by Kyuran, the author of Croco DS and PenkoDS.

The very first release 0.1a is able to run most games at 60% speed allowing you to enjoy Xenon 2, Rick Dangerous, David Braben's Virus and many more...

some impressive pics (made by Lord Raptor) :

http://img442.imageshack.us/img442/5052/dmue2.jpg

http://img262.imageshack.us/img262/7671/populousox4.jpg

http://img262.imageshack.us/img262/6987/skweekfe9.jpg

01a features (rough translation from author's website)

- .ST files support
- sound
- DLDI support

How to :

copy Atari ST TOS and .ST files in the root or /roms/st directories of your card, patch the .nds with DLDI and... that's it! :-)

Tos 1.xx and 2.xx should run ok, but note that I used v1.02. Make sure to rename the file to tos.bin

Don't forget this is an early alpha release. There are many bugs, I'm quite aware of that.

Please don't ask for roms and games. Google is your best friend. You should be able to find all Automation releases just three mouse clicks away from the Steem Engine website, hint hint ! ;)


Author's website : http://deadketchup.kyuran.be
File : http://media.kyuran.be/temp/styxds-dldi-v01a.nds

Some roms start automatically.
Some of them bring you to the green desktop. From there, double click on the disc volume (on the top left corner) with the touchpad.

    Sometimes you have nothing, and I suspect bad rom, emulator bug or compatibility problem between the TOS version and the rom. Try another rom then.
