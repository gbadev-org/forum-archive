Hello, everybody.

You certainly know my Sprite editor "SEDS", which has been around for a while and updated lately. Well, here's some good news if you had drawn some game art with it: I have at least a first release for its companion Level Editor on DS -- aka LEDS.

The software is still quite rough on the angles, and is more an internal tool released to the public, so expect some learning to be needed. But if you love doing some level design here and there and if you have empty time to fill in the bus or during step-family evenings, LEDS is the tool you'll love.

- up to 512x512 tiles size for the map (or any other
- two-planes and separate layer for tiles attributes
- native flipping support
- no conversion needed in your game engine
- open source, and 100% compatible with the open-source GEDS game engine.
- shipped with two demonstration tileset available on OpenGameArt
- online tutorial to get you started quickly

enjoy Level Editor for DS 0.1
