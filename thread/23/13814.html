#136310 - quinbd - Tue Jul 31, 2007 3:26 am

OK here is how to make it work.

1)Unzip the file
2)double click the server and click "Start!" on the server window
* Note server must be running while the app is running.
3) Start->Run type "cmd"
4) type "ipconfig" look at "IPv4 Address" that is your computers IP
5) Start the app on the DS
6) when it asks for the server, enter the IP of you computer
7)Hit enter, then enter again unless you changed the port number on the server window
8) type web address and surf!!

*Troubleshooting
1) Unless you have a static IP your computers IP will change every time you restart the computer.
2) You may need to port forward port 666 to the ip of you computer. To do this look here.
3) And you might have to allow the program or port through a firewall. But it worked fine with the windows vista firewall.

Good luck!
