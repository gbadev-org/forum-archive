I've made the "edge-matching" puzzle game Tetravex for the DS. :) Here's the current features:

- Different board sizes: 2x2 to 6x6
- Numbers are coloured like the Gnome version (gnotravex)
- Hints
- Can move all tiles at the same time with the D-pad

It's a bit weird to play sometimes because I had to put one grid on the bottom screen and one on the top. So you have to switch screens (using R or L) whenever you want to put tiles from the top screen back to the bottom screen.
