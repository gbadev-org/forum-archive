you guys, when you get a bugged screen, you need to start pushing buttons..like B, or A.. it will update the screen and will show graphics..that will show that it's working.

It's a new program, there's bound to be problems with it.

Here are things to make sure of:

Make sure server.exe is running
Make sure you are not using WEP on your wifi network
Make sure you are directing the destination IP to the right computer, your computer with the server running

There are many factors to consider..If you got the past Wifi programs working, setting this up is a breeze.
