#143131 - Dood77 - Wed Oct 17, 2007 5:31 am

If you downloaded the video from youtube and the extension is .swf, then you downloaded the wrong file. It should be a .flv file. If you're having trouble with .flv files, I would ask if you have BatchDPG installed correctly.

If you're trying to comvert an actual .swf file, the kind that will play in your browser, then you can't do that in BatchDPG. You'll need some other program to record the .swf file into an actual video file.
