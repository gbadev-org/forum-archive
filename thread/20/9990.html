#88645 - tepples - Tue Jun 20, 2006 7:26 pm

Linkiboy wrote:
but I hear GBAMP doesn't support GBFS. Is that true?

GBA Movie Player v2 CF supports everything but the find_first_gbfs_file() function. You'll have to load your GBFS file into RAM using one of the following methods:
  1. (for GBAMP and FlashMe+WMB) Include the .gbfs files into the program using bin2s when you build the ROM. This will require you to distribute at least the object code, the bin2s program, the assembler, and the linker.
  2. (for GBAMP and FlashMe+WMB) Include a .gbfs file containing a large zero-filled file into the program using bin2s, and then have your builder overwrite the data starting at the "PinEightGBFS" signature.
  3. (for GBAMP only) Load the .gbfs files from the CF card into RAM using Chishm's FAT driver.
