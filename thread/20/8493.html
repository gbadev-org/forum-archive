MoonShell Skin Pack Release 1.0

Hey folks,

I was messing with different MoonShell skins and I've finally made up a Pack in which all the skins are I found.

Just download MoonShell Skin Pack here and install it on the root of your SD Card.

It is tested with SuperCard SD + miniSD 512 MB + WXP SP2

All the skins are zipped so everything is very small. ~1,6 MB.

It puts a small file PKUNZIP.EXE in shell/skinunzip.
The Skins are put into the folder "Skin Pack".

To install a skin just use the install.bat.
It is the same as Installieren.bat just that install.bat is english and Installieren.bat german.

Following files are automatically removed from shell:

c_file.bmp
c_text.bmp
client.bmp
desktop.bmp
prgbara.bmp
prgbard.bmp
sb_bg.bmp
sb_body.bmp
sb_part.bmp
titlea.bmp
titled.bmp
skin.ini

For the other files included in the skin as e.g. startup.mp3 there will be a prompt later because you might want to keep your startup sound.

Have fun and if anybody has questions (I bet 100 euro there will be :) ) just post them.
