#85523 - TJ - Wed May 31, 2006 9:11 am

For that, he would need two computers though, since the Linux drivers for the Nintendo Wi-Fi Connector (or any RT2500 based card, for that matter), do not yet support Master mode, and can't be used as soft APs.

So he would need the one XP machine to actually be used as the AP, and then route that machine though the Linux machine, which would actually do the capturing.

#85597 - TJ - Thu Jun 01, 2006 4:58 am

An AP can only set its data rate as high as the devices connected to it can run at.

So even if you have a G router, the DS running at 2 Mbps will bring all the traffic down to that speed when it is connected.
