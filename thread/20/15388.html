Eepinator can dump the save files of games that use SRAM to save, though I've heard you have to dump it twice for it to work for some reason.

There is another program that works with more save files, but as it is (I believe) also capable of dumping commercial GBA cartridges it's probably best not to mention it.

If you have a DLDI-capable slot-2 device, you can use Chishm's cart_save program.
