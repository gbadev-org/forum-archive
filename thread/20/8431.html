I'm simply including the text files in the IMFS until the issue with reading them off the card is resolved.

In the moonshell folder, under "files" I made a subfolder for my ebooks, slapped them all in there, and then went up one level to run WriteFiles_IMFS.bat. The shell script generates a new _BOOT_MP.nds in the main folder to be dropped onto the card.

The text files are included in the .nds binary, which counters a lot of the convenience of being able to use a CF card, as one has to update the "files" folder and rerun the script upon each change. As a temporary workaround, though, it's more than bearable.
