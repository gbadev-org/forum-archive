#98624 - Mota - Wed Aug 16, 2006 3:52 pm

I had several 'issues' with the Max Media Launcher - not only did it feel 'too big' for the slot, it often wouldn't boot at all, and sometimes required a little pressure from the top to make it work.

Worse still, on a couple of occasions it crashed in some way, making my DS appear to be bricked. The ONLY way i could get it to work again was to remove the battery from the DS for a few minutes, then try again. I assume it was leaving crappy code somewhere in the DS's memory.

I would try the battery trick first, and if that doesn't help, attempt to flash it with Flashme7-Stealth like Tepples mentioned. If there's a hardware fault, and you're able to install the stealth version, Nintendo will probably replace it for you. They did for me.... heh
