1. I first insert my CF card into the reader (SD card should be no different here).
2. It automatically detected it in WindowsXP and assigned it as drive "F:" (I would hope that Linux would do this too).
3. Then I ran Setup.exe and chose it to write to drive "F:" (if your card isn't assigned as a drive letter, try selecting "C:" as this will put the "_BOOT_MP.nds" file at the root of the drive and create the necessary "\shell" folder at "C:\shell\".
4. I left "/shell/global.ini", "/shell/desktop.bmp", and "/shell/startup.mp3" checked as those get copied from the "\SetupData\files\shell\" subfolder from where Setup.exe is running (you can also replace those files with different ones as long as they carry the same names and the desktop.bmp file must be 256x192 and a 256-color indexed palette).
5. Since I didn't need any additional language settings, I just unchecked the box to copy a new one over.
6. I also unchecked the SuperCard option since I have a GBAMP. You might want to make both files and see which works better.
7. Clicking OK wrote everything out to the CF card without a problem.
8. For the skins, I just unzipped one of Moonlight's skins and copied the "skin.ini" file and all of the bitmap files into the "\shell" directory on the CF card.
9. Replaced the CF card in the GBAMP, put it into the DS, started it up, and viola! Skinned Moonshell v.99!

1. I first insert my CF card into the reader (SD card should be no different here).
2. It automatically detected it in WindowsXP and assigned it as drive "F:" (I would hope that Linux would do this too).
3. Then I ran Setup.exe and chose it to write to drive "F:" (if your card isn't assigned as a drive letter, try selecting "C:" as this will put the "_BOOT_MP.nds" file at the root of the drive and create the necessary "\shell" folder at "C:\shell\".
4. I left "/shell/global.ini", "/shell/desktop.bmp", and "/shell/startup.mp3" checked as those get copied from the "\SetupData\files\shell\" subfolder from where Setup.exe is running (you can also replace those files with different ones as long as they carry the same names and the desktop.bmp file must be 256x192 and a 256-color indexed palette).
5. Since I didn't need any additional language settings, I just unchecked the box to copy a new one over.
6. I also unchecked the SuperCard option since I have a GBAMP. You might want to make both files and see which works better.
7. Clicking OK wrote everything out to the CF card without a problem.
8. For the skins, I just unzipped one of Moonlight's skins and copied the "skin.ini" file and all of the bitmap files into the "\shell" directory on the CF card.
9. Replaced the CF card in the GBAMP, put it into the DS, started it up, and viola! Skinned Moonshell v.99!
