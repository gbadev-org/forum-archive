My vision (for what it's worth) for viewing multimedia files on the DS which contain both text and images - like PDFs and HTML pages - is to use Alt text to display images. When the Alt text is clicked, the image is opened as a full page. Pressing a slightly more prominent than usual Back button (or double clicking the image?) would bring us back to where we were.

Thus, the images are all singled out and viewed by explicit request of the user so we aren't viewing hundreds of images, text, and videos at the same time.
(Perhaps smaller images could be displayed automatically based on RAM availability...).
