#56894 - tepples - Wed Oct 12, 2005 5:08 am

Interlace artifact lines are normal on Nintendo systems with color LCD panels, whether GBC, GBA, or Nintendo DS. The system draws every other scanline darker in alternate frames in order to make pixel response time appear faster. If you can run GBA programs, go get the "static" demo from here and run it on each screen.
