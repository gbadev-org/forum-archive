#70391 - Joat - Mon Feb 06, 2006 2:21 am

If you do *not* have flashme installed, then there will be no problems at all with running mario kart.

If you *do* have an older version of flashme installed (installed before last November), then you should upgrade again before running mario kart.

If you run mario kart on a DS with an old version of flashme, it is not as bad as failing a flashme install. The failsafe is still in place, and you re-run flashme from a GBA cartridge by holding down A+B+select+start.

If you choose to run flashme, PLEASE PLEASE PLEASE read the instructions very carefully and follow them. If you keep your DS plugged into the wall, then you can leave it on for as long as you need to, until you are able to properly short SL1 again. Never turn off the DS if the flash has started at all.
