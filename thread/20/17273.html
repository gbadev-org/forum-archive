DS Misc > Convert DSi XL from English to Japanese?

#174211 - catphive - Mon May 24, 2010 1:54 am

Some background:

I have a number of japanese cartridges, and used them on my original nintendo DS to practice japanese. However, my old DS died, so I bought a DSi XL.

When it arrived I realized there was no way to put it in Japanese mode, like I could with the original DS. This is ok for monolanguage cartridges which still show japanese. However, for instance my Chrono Trigger cart is actually dual language, so the DS language setting being set to English forces it to run in English mode every time.

Is there any way for me to change the dsi firmware to use the japanese firmware, or in some other way to enable the use of japanese language? I really just need the DS to tell my cart to use japanese, I don't need access to the DSI store or whatever.

If this is not the best place to ask, let me know where...

#174219 - SifJar - Mon May 24, 2010 2:31 pm

For regular DS (i.e. non "i") games, there's something you could try:

Get a DSi compatible flashcart
Run the original NDS firmware dump thats floating about the internet via the flashcart
Change the language in it to Japanese
Restart DSi and run DS game

This works for pictochat and download play which run in DS mode, so it may work for games running in DS mode as well.
