#115650 - dantheman - Thu Jan 18, 2007 3:51 am

All DS emulators have flaws even this late in the game, and are usually not a good representation of how homebrew will run on the DS. For instance, tepples's AXE demo freezes No$GBA (the latest versions have DS functionality), while the same demo runs well in Dualis. I don't think any of them run the recent nightly builds of DSLinux, though an older build worked fairly well in some other ones.

Others you can try include iDeaS, DSEmu, GoodNDS, and desmume.
