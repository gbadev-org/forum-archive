The problem with porting it is the fact that there is no source code to TA available. There is an open source project called TA Spring, but it is so much more advanced than TA, that a port would be very difficult if not impossible. What WOULD be nice would be a C&C open source engine (like the Vegas engine is to the Wing Commander genre), and have it ported over to the DS. I believe I saw one, but it's been a while. A quick glance at sourceforge.net and I ran across numerous different projects in that type, including FreeCNC, an open source reimplementation of the original Command & Conquer. I also saw several RTS game engines that looked nicely suited for the DS, but as I'm not really a programmer, I wouldn't know how well it would work.
