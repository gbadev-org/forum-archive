#60778 - sgstair - Mon Nov 14, 2005 7:21 pm

Dudu, please note the following:
in the W3C specification for HTML 4.01, it says:
"This specification does not define how conforming user agents handle general error conditions, including how user agents behave when they encounter elements, attributes, attribute values, or entities not specified in this document."
in an IMG tag, the alt attribute is considered required, for accessability reasons (I don't always implement it on my site tho)
However, the title tag is not considered to be required, and what is done with it is left up to the implementor.

Technically, neither is wrong, because HTML 4.01 does not specify what must be done when an element (in this case title) does not exist.
Using the alt as a tooltip is a perfectly acceptable response to not having a title attribute.

So, please don't state definitive information on things you don't know anything about.
