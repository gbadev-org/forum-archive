MoonShell (for Nintendo DS, by Moonlight) (Website: Google translated: English | Original language: Japanese)

Release notes:
test4-dpgenc2 2006/02/13
A synchronous problem is corrected...?
Please overwrite onto dpgenc.exe of MoonShell0.992test4.

test4-dpgenc 2006/02/12
The mode to be able to acquire the voice by using mencoder or ffmpeg was added.
Please overwrite onto dpgenc.exe of MoonShell0.992test4.

test4 2006/02/12
The maximum reduction rate from 65% to 50%.
MP3Plugin:A reactive speed of the seek operation has been improved.
DPGPlugin:'mpeg1-audio-layer-2' can be used for the voice codec.
sorry. The voice codec means audio codec.
The tool related to DPG works only in the WindowsNT system. (Win2k/WinXP etc.)
The DirectShow codec recommends ffdshow very much. (I use ffdshow-20050909.exe package. )
Attention. There is considerably what to the condition of OS worsens when the codec is installed.

test3 2006/02/10
include TouchPad calibration tool for MagicKey2/3.

test2 2006/02/10
little change for FAT driver.
old SCSD assembler code include.
change auto-detect priority. (NMMC->M3CF->MPCF->SCCF->M3SD->SCSD->FCSR->NULL)

test1 2006/02/09
The FAT library was updated to gba_nds_fat_2006-02-09. (very very thanks Chishm)
There is a problem in the touch panel when MagicKey2/3 is used. (Strange?
A small bug to shuffle was corrected. (Thanks 681)

Version 0.992 2006/02/08
Re-make oversampling code.
SCSD was changed a little. (Thanks cory1492)
The movie encode option of dpgenc can be edited.
Use library was renewed to libnds-20060201.
readme.txt of German was added. (Thanks Ohr GmbH)
The bug that MusicNext hangs up was corrected. (Thanks 679)
The ARM9 interrupt code was moved to ITCM.
When failing in the file open, information is displayed.
--------------------------------------------------------------------------------------------------------------
Filters: ffdshow 2005/11/03 Download: Installation file

test4-dpgenc2 Bugs:
- MoonShell says that the time is 11:03:00 while the real clock says it is 23:03:00

Wish list (or am I just dreaming):
- Backlights on when lit is opened.
- No blinking green light when standby. Only blink when lit is closed.
- Possibility to lock (freeze) DS shoulder buttons in the MoonShell.
- MoonShell shutdown if battery is running low when lit is closed.
- Files and folder are counted like: "3 files". Better to count them like: "3 files/folders".
- Ability to change date order. Example: dd/mm/yyyy, yyyy/dd/mm and mm/dd/yyyy (d=day, m=month, y=year)
