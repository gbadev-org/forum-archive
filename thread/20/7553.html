#63455 - DiscoStew - Tue Dec 13, 2005 7:54 am

mymateo,
hehe, I'm not much with networking either, but once you starting tinkering with things, it starts to make sense.
What I did was I went all out on pieces that MIGHT work. Here is what I did for my router (don't think it will be the same for yours, but worth a try if you can find all the things)

On router settings...
- Set a reserved IP address for the DS. (You've done this with making sure the MAC address is the same as on your DS?)
- Set that IP address as the DMZ server (You did this too)
- Disable SPI firewall (if you can).
- Port Forwarding:
--- Set UDP ports 1 to 65534 to your DS IP Address (If you have other games using specific UDP ports, you may have to work around the used ports)
--- Set TCP ports 28910 to 29920 to that IP Address

On the DS...
(Most likely your router has SSID broadcasting enabled, so by Searching for Access point, it will find your router, filling in the SSID, and, if it is requiring, enter the WEP key)
- Enter your static IP Address like you did on your router
- Subnet Mask: 255.255.255.0
- Gateway: 192.168.0.1 or whatever you use to get into your router settings
(By now, "Auto-obtain DNS should be set to No)
- Primary and Secondary DNS: These can be found under your Router settings under "Domain Name Server". Some people report that using 4.2.2.2 and 4.2.2.3 work, but for me, they don't

Hopefully that helps.
