Another awesome emulator by AlekMaul

Version 1.0 has been released :

- language selection
- DLDI support (patcher and drivers are included)
- copy emulator and games (GG or SMS) in a DSMASTERP directory in the root directory of your card.

Grab it on the author's website : http://www.portabledev.com/pages/ds/jeuxdev.-perso/dsmasterplus.php (file is at the bottom of the page)

My little bit of testing :

- Game Gear games run faster than Sega Master System ones. Columns and Pengo are even running at full speed with perfect sound!

- Saves states work but you can't load them. AlekMaul stated that this will be corrected in the next release.

#120080 - Metaluna - Wed Feb 28, 2007 10:41 pm

Alkemaul has just released Version 1.1! (file at the bottom of the page)
tons of improvements :

- Use L and R in the game list to swap through the files faster
- B button to exit a directory
- Directories navigation added
- GG games in SMS mode added (for games such as Castle of Illusion)
- GG and SMS files have different colors in order to be easily recognizable
- English / French menu depending on the country selected in the DS menu
- CodeMaster and Korean mappers added
- TMS compatibility mode
- Speed has been increased. The cpu is now being asm emulated by DRZ80.
- No more save states (Alekmaul is working on it for the next release)
- German language added (thanks Payk!)
- Compatibility report can be send by WIFI to this page :
http://www.portabledev.com/pages/ds/jeuxdev.-perso/dsmasterplus/liste-de-compatibilitE9-des-jeux.php
