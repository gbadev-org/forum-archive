Is there any reason you're stuck to using .xhtml? Granted it's the format of DSLibris, but there are a few other readers available. DSReader is an EXCELLENT txt reader, that only lacks rotated reading support. ReadMore Alpha is an excellent alternative to DSLibris that just views txt files (unless you're one of the rare few that has the unfortunate topscreen off bug).

In the case you can use any of the many .txt viewers, there is at least one good .pdf to .txt programs out there. PDF to Text. (don't go to the publisher's website, as their link is faulty and sends you the Free PDF to Word Doc converter - also a good tool, but not for this purpose).

Wonderfully, PDF to Text Converter converts .pdf files into .txt files very quickly and can do any number of pdf files you have in a folder. It unfortunately, however, creates a .txt document for every page. It also only loads and saves from a single folder, so it has some limitations, and the quantity of files IN that folder can become somewhat daunting. This is easily fixed, however, with a pretty basic Command Prompt .bat file.

Create a Combine.bat file and copy and paste this code into it and save it.
Code:
FOR %%1 in (NameOfFile.txtNameOfFile_PG_????.txt) do type %%1 >> NameOfFile.txt

Change all instances of "NameOfFile" to the name of the ebook file you wish to combine (without the .txt extension).

Now, given a bit more time and impetus, I can create a few .bat files to automate this process so much more nicely. I'm already hashing out some code for the PDF to .JPG converter as I have some ebooks with a LOT of pictures in them I'd rather be able to see in their original style.

#157583 - dantheman - Mon May 26, 2008 11:21 pm

I personally use Paperless Printer for converting PDF to JPEG. Simply open the PDF file, do File - Print, choose Paperless Printer, and then choose JPEG from the screen that pops up. No watermark, and it works well. From there you can convert to ComicBookDS format.

I know the latest version of the PicToDS converter for ComicBookDS has a menu option to go straight from PDF to CBDS format, but I never understood exactly how to get it to work.

If you want just plain text, I got rid of the "?" characters in Moonshell and DSReader by opening the text file in Wordpad and saving under a different encoding. I believe I saved it as "Text Document - MS-DOS format" after which point it worked fine. Note that DSReader's bookmarks only work if the filename has no spaces.
