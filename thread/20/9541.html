#82888 - Flood_of_SYNs - Thu May 11, 2006 1:06 am

On my desktop I have a batch file for optimizing PNG images, you might be able to do something similar with compressing video.
I just drag the unoptimized PNG onto the batch file and and it does the rest from there.

png-compress.bat wrote:
echo off
title %1
d:
cd optipng-0.5.1-exe
optipng.exe -zc1-9 -zm1-9 -zs0-2 -f0-5 %1
pause
