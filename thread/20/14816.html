Just a quick note for using this: you need to make a "lmp-ng" folder on the root of your card, create a "skins" folder inside that, and add at least one skin (left in .zip format, don't extract them). Otherwise the screen will show up blank, though the buttons will still work if you can navigate without seeing what you're doing.
