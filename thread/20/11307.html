#104578 - tepples - Sat Sep 30, 2006 5:03 pm

  • The GBAMP v2 CF is a NOR flash (for firmware) and an ATA interface (for CF).
  • The MAX Media Dock is just an ATA interface, as the firmware is stored partly on the MAX Media Launcher and partly on the CF card.
  • The MAX Media Player is a MMD with a CF hard drive.

The GBAMP and MMD ATA interfaces are placed at different parts of the GBA ROM address space, and the block device driver used by a FAT library needs to be modified to take this into account. But this change is very easy to make, and usually all the author of a program has to do is update to a new version of the FAT library that already has this change in place.
