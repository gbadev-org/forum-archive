All routers use a web page for configuring settings.(Like setting/retrieving WEP)
The only way to access this, is to know the router ip, and the user name and
password.
You might be able to find WEP, if you dig around in Network Connections.
(I don't know if this will work; The only hands-on experience I had with
networking was with these ASUS drivers. That didn't work out and now I'm hunting for a WiFi device for SoftAP)

You might need to tweak some settings on the router to get DS/Wii to work
with it, if those settings aren't set correctly already.(Like transmit rate, and 802.11 b standard)

You're going to need this connector, or a SoftAP if the router won't work with DS/Wii.
