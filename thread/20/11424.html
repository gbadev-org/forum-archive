#105960 - OOPMan - Fri Oct 13, 2006 5:54 pm

DS Opera does not use proxy servers in the same way that Opera Mini does MiL0...

Opera Mini recieves all it's data from a proxy that pre-processes everything...

DS Opera is able to make use of a user-specified proxy in much the same way a normal browser can.

For example, DS Opera from Japan includes details on setting up some kind of proxy that looks, by the images etc, to be a filter that will stop young users from accessing naughty sites :-)

I doubt you'll see any speed improvements in the european version, unless you're running some kind of funky proxy that mimics the behavour of the Opera Mini ones (This wouldn't be a bad idea in some respects, considering the shoddy performance of DS Opera...)

When using Opera DS I recommend turning of image loading...

It boosts general speed about at least 200%, probably more.

Also, for forums and stuff images are extraneous anyway...

You can also boost speed by turning the SSR screen mode on, but I prefer image-less DS mode, since the zoom feature is really useful, while SSR is actually more difficult to navigate...
