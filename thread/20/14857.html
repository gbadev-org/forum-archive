#149190 - M3d10n - Wed Jan 16, 2008 2:59 pm

Not possible. Even if it could fit, the button placement is slightly different, so the contacts would not batch the holes in the case. The motherboard hardly uses any wires, so other IO stuff like volume control, headphone jack and the charger plug are all directly soldered on the motherboard, so those wouldn't fit either.
