Hi there!

I have been developing a DS homebrew game using DS LUA: Snowball battle.

Update: New version v1.3 released (now with music)!

Game description:
A winter snowball battle in your DS! Four players loaded with some snowballs in a snowfield. No much explanation required: try to hit the others throwing snowballs while dodge theirs. Two out of the four players of the game can be humans playing in the same DS!
