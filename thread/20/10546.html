Unfortunately Scummvm DS currently does not recognise the Max Media Dock as of right now. Chishm has released a new libfat that should include compatability for the Max Media Dock so that it would allow scummvm to run on it when it is implemented. The setup of the Max Media Dock wouldn't have it use the zip method anyways though as that is for GBA flashcarts. I actually got into the menu using my Ipod's HDD but it doesn't show the files (and therefore can't run games). I guess we just have to wait for the next version.

You can read my story about ScummVM and the Max Media Dock at:
http://forums.maxconsole.net/showthread.php?t=25018

P.S.- Don't try to tear apart your Ipod. The HDD does not work well with the MMD.
