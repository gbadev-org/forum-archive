#141331 - dantheman - Mon Sep 24, 2007 4:36 am

Depends on what the slot-2 device is. If it's the EZ-V 3in1 expansion pack, something like that called NDStation already exists. Rudolph's tool can also load GBA binaries to the 3in1's RAM and launch them. The Supercard DS has a built-in function to run GBA binaries when a non-Rumble slot-2 Supercard is inserted, and I believe the CycloDS Evolution can do the same when paired with a slot-2 CycloDS product.

If you're referring to the Opera RAM pack, I don't think it's possible to execute GBA binaries from it, though I could be wrong.
