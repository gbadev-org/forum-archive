#154741 - dantheman - Mon Apr 21, 2008 4:05 am

sonny_jim wrote:
Blatant plug, but if you have a RAM pack that's supported, I find the text mode browsers (links and retawqs) are pretty handy once you get used to them. I wouldn't bother using the graphical mode in links as it is still quite buggy.

You could set non-graphical Links to show [ IMG ] where the image would go on the page, then click on that to download it to your card, then use graphical Links to view it. At least that's what I do, which is helped immensely by tab-completion.

Without slot-2 RAM, look into Bunjalloo and possibly DSHobro if you're fine with leaving a server running on your computer all the time.
