Cool, my favorite board game Go on the DS! This shows potential. I played a rather quick game against the CPU, which was quite fun already.

Here are some suggestions:

Stylus support would be cool, but I know it's hard to write accurate touch code for such an app. Perhaps it will be easier if you make the board bigger. The AI is pretty weak, in fact it is really dumb. Is there a way to use perhaps a newer version of GnuGo or increase the AI level? Also, I would like 9x9 or 13x13 boards for quicker games.
At the end of the game, there was no possibility to remove dead stones and the final scores were not displayed, only the number of captured stones. And there should be a message like "Black wins".

But anyway, this could become great game! Keep it up!
