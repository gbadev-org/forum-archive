#80288 - quadomatic - Fri Apr 21, 2006 1:56 am

El Hobito wrote:
well what advantages does it have? blogspot seems pretty impressive and does what i need, the only thing i really lack is some where to host my files


Well, Wordpress just looks a lot cleaner and quite a few features that would have to be coded manually in blogger are already there in wordpress. Wordpress's themes aren't quite as customizable as blogger, but there really isn't a need for that in wordpress cuz its got so many features to start with.

Also, you may get annoyed later with blogger because spam in the form of comments gets onto your blog. Wordpress allows you to moderate comments and I believe it's capable of checking those kinds of things itself.

Wordpress also automatically pings pingomatic.com

And, don't worry if you do use blogger for a while and decide to move from blogger to wordpress. You can import everything from blogger to wordpress automatically. I haven't tested how well that works though, but it should be fine.
