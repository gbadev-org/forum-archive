#135898 - dantheman - Fri Jul 27, 2007 2:56 am

There is no web browser specific to the R4, and it does not include one built-in. The only way to run the official Opera browser is to buy it from a store. And yes, you can get the Phat-sized version though I think it's only available online for US users.

There are a few homebrew web browsers that will work, though most are text-only. Check out DSOrganize, Bunjalloo, and DSHobro, and also consider trying the Retawq web browser in DSLinux.
