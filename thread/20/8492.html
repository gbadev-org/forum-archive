With most homebrew all you need to do is to chop off the first 512 bytes of the file (that'd be where the loader is)...this will usually work, unless the program uses a custom loader (like a lot of older stuff probably would, maybe some stuff that requires a GBA flash cart would as well?).

Search Google for File Trimmer, it'll do this very easily.
