#106764 - Sektor - Mon Oct 23, 2006 6:56 am

If you are using the nintendo usb wi-fi connector then you will need to use unofficial drivers to change it into a softAP since homebrew isn't compatible with the nintendo drivers.

Just because official games work doesn't mean homebrew will. For some people DHCP fails with homebrew, so you need to enter a static IP.

If you entered your WEP key as ASCII then you need to change it to Hex. Also try without WEP just to see if you can get it working. Make sure broadcast SSID is enabled or hide SSID is disabled.
