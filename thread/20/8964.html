#76269 - carmex - Sun Mar 19, 2006 11:16 pm

I've released DSMerlin 0.3. All major features are now included. Future releases will either be minor bug fixes or concentrate on reducing ROM size. As always, grab the ROMs or source at: http://ds.crmx.us/dsmerlin/. Changelog:

March 19, 2006
version 0.3:
Library changes:
* Implemented alpha shift and caps-lock.
* Implemented punctuation shift mode.
* Added most common punctuation and symbols.

DSMerlin demo change:
* Added shift and caps-lock to alphanumeric help screen.
* Added punctuation help screen (press A twice to view).
* Implemented "crossover" shifting. Cross the divider line when drawing a letter to automatically capitalize it. Thanks to Medha for this suggestion.

#76685 - carmex - Thu Mar 23, 2006 4:01 pm

DSMerlin is just a library that others are free to use in their own projects. I won't really be developing a gui for it since that's the job of the application developer to make a gui that fits with their app.

Drawing your own letters and symbols WOULD be a very neat feature. You could even set up your own gesture-based macro system. But that would involve reading and writing files to the flash card (to save the patterns of the symbols you drew). Not impossible by any means, but I'm not ready to tackle that just yet. :)

Also, adding support for symbols in other languages is no problem. I'll be happy to do it once there's an actual need. As of now, and AFAIK, the only program to use DSMerlin is my own demo and poffy's poffPad, and I think he has stopped working on poffPad. So if anyone else decides it's worth using in their app I'll put more time into it, but it's not really worth developing further if no one is goign to use it. :)
