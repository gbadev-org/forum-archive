#99121 - MaHe - Sat Aug 19, 2006 5:06 pm

Run the latest DSOrganize, open up the Browser, select "New File", name it to 'whateveryouwanthere.pkg.txt'. Now edit it like that:
Code:
chdr /@down <yoururlhere>@
For example:
Code:
chdr /@mkdr Concerned_Comic@chdr /Concerned_Comic/@down http://www.hlcomic.com/comics/concerned000.jpg@down http://www.hlcomic.com/comics/concerned001.jpg@
And remove the .txt extension using the 'Rename' command. Then just run the file and it should work fine.
