#135202 - Dood77 - Fri Jul 20, 2007 10:29 pm

calcprogrammer1 wrote:
I tried it on my regular DS and it ran great...it picked up more AP's than my laptop and PDA combined could :)

This is because the wardriving test picks up any stray packets from any source, disregarding if its viable to actually connect to the access point or not. Default software on PCs only display access points that they can actually get packets from on a regular basis, so its possible to do all the DHCP mumbo-jumbo to actually connect to them. Wardriving software on the PC however, will pickup as much if not more than that test on the DS, with more specific information, no doubt.
