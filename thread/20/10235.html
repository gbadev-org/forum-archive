#91277 - The 9th Sage - Thu Jul 06, 2006 12:05 am

leirch wrote:
I've tried every possible boot file. I'm not sure whats going on. Could it be a problem with the M3 itself?


Try other homebrew...both NDS and ds.gba style homebrew. If they work then I'd guess that it's not the M3 itself, unless MoonShell is having an issue with SD M3s for some reason, but it shouldn't be that I know of. I'd probably try reformatting the SD card with FAT32 if it's FAT16 (FAT) or FAT16 if it's FAT32, just to see if that helps.

*edit*
Also, if none of this works, try a different SD card if you can. It's possible it's having some weird compatibility issue...I've heard of this happening with some SD cards and the M3 SD. Also make sure the M3 is flashed with it's most current firmware and see if that fixes it (you can always 'downgrade' if you wish). I wouldn't think it'd be an issue, but I seem to remember some strangeness with old M3 firmwares and the DS Lite. Never experienced it myself, so not sure what that was about.
