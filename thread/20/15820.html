#160387 - yellowstar - Mon Jul 14, 2008 11:05 pm

Since you have a router, you should be able to do Wifi on DS fine. The router needs to support 802.11b,(802.11g should work also, since that's backwards compatible with b)
and WEP.(WEP isn't really required, but if you want the network to stay secure, it's needed.)
If your router doesn't support WEP, and is using some other security, well, you'll need a Wifi dongle. Unless opening up your router with no security is an option, and that's probably not an option.

If official DS games with Wifi work with your router, it should work fine with homebrew. But if you have a DS Lite, some homebrew may not work unless the router's channel is set to channel 11.
