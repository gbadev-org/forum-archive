V.51 is out!

Hello all!
Let me explain what this program is.
This program allows many people without coding experiance to create their own DS RPG games, similar to RPG Maker 2003 for PC, but this is for DS!

Early early versions have come out in the past, but only allowed for you to make maps and play on them as a little square guy. But, I have been following development of VGMDS and VGRDS (Virtual Game Maker DS) and it has progressed RAPIDLY since the last release!

You are now able to import your own graphics, create events, set hero start position, have music play in the background, and much much more!

I am the main tester of VGMDS, and a public release has just happened.
It is a very very important release, a breakthrough you might say.

Here is a quote from the great Globoeil himself
Quote:
Updates in v.0.50:
===================
-Added custom chipsets
-Added events & event system
-Added multi-maps support
-Added custom charsets support
-Acclerated loadings (in VGXDS and when starting the programs)
-Added system(custom fonts in game)
-New File Explorer v2.5
-New interface (easy to use)
-Added "move event" part
-Added "Commands" part
-Added "Gfx" and "Conditions" part
-Added advanced event manipulations (Copy, Paste, Cut)
-New instructions:
-Quick message
-Move Up/Right/Down/Left
-Play Sound
-Teleport (support multimap perfectly)
-Message (fully customisable, size, position, transparency, lines etc)

-Improved event collision in VGRDS
-Improved game fluidity
-Improved Blackscreens when teleporting or loading maps
-Added starting event command conditions(pressing A,on touch, on screen, Auto-Start)
-Added power saving functions (when DS closed)
(There is a lot of minor updates not described here)
