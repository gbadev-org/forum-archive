#80122 - Mr. Picklesworth - Tue Apr 18, 2006 11:46 pm

It's unlikely to happen for a while (or ever), since a drawing plugin for Moonshell would require significant additions to Moonshell's plugin system.

For one thing, it would need to be able to boot plugins at any time just like it boots anything else.
I'm also unsure as to what kind of control the plugins have over windows, and how they deal with user input... a good drawing app in Moonshell would probably want to create two seperate windows for itself, and would need some pretty interesting UI stuff which Moonshell currently doesn't do.

I'm willing to be surprised, though :)

#80466 - tepples - Sat Apr 22, 2006 2:22 pm

tssf wrote:
VLC and Media Player don't play MOD, SPC, and any other file formats that would require a custom plugin, do they?

"Media Player" is a generic term. Call it Windows Media Player or WiMP. Compared to other media players, WiMP comes with few codecs, but a lot of codecs (called "DirectShow filters") are available for download at no charge on the Internet.

VLC media player comes with lots of plug-ins, including one based on ModPlug Player for MOD/S3M/XM/IT, but not one for SPC (Super NES audio rips).

#82194 - chishm - Thu May 04, 2006 2:57 pm

tepples wrote:
Does a file system have to be unmounted before being removed, or can it just be synced? And what's the best way to be notified on cartridge eject/insert so that MoonShell can unmount/remount the file system?

Unmounting is only crucial if you wish to remount. When you close a file, it's contents are synced to disc and all buffers are flushed, so there isn't a risk of data loss. However, when you unmount, all files and the hardware interface are closed. If you don't unmount, it can't reopen the hardware interface and you won't be able to remount.

There is currently no way to be notified of a cartridge removal, asside from using cartridge interrupts. However, once that happens it is too late to unmount.
