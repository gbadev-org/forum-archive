#106134 - gardevor - Sun Oct 15, 2006 7:07 pm

I'm not sure about Pokemon..but here are the steps I took to get Castlevania DS (jap) working:

1. Download Dipstar 5.11 or 5.12ff

2. Write dipstar.ds.gba to G6 as a GBA rom with the game manager with no options ticked.

3. Drag and drop your ds rom onto m3dip.exe (located in the m3dip folder of the extracted dipstar package), a DOS window will pop-up and press Enter.

4. Write your dipped ds rom to your G6 with the correct settings that works with the game.

5. Turn on NDS, goto MyCard>GBA folder and scroll down to Dipstart.nds.gba, Press X button ONCE.

6. Go back to NDS folder and load the desire game then quickly press and hold the L button.

7. Dipstar should load.

You have 8 slots to enter codes. Each slot is for each button (A,B,Start,Select & directional)

Press the desire button where you want to save to codes in then enter the codes.
Press Start to save the code.

Reboot your NDS.

Goto MyCard>GBA folder and scroll down to Dipstart.nds.gba, Press X button ONCE.

Go back to NDS folder and load the desire game then quickly press and hold the button where you saved the codes in (ex. if you saved the codes in directional UP, then hold UP while the game is loading).

You can try the cheat codes in the G6 installation folder first, just to make sure everything works.
Hope that helps!
