rsreston wrote:
I have a D-Link WiFi USB but I cannot access the internet with my NDSlite - does anyone know how can I setup an access point with it?


Check this thread

When you have H/W version B1, you can use a patched version of the original Nintendo USB Stick software.

(The H/W version is noted on the back of the USB Stick)

Otherwise (if you have revision C1) it depends, if the original D-Link Software allows SoftAP (Software Access Point Modus), use it, and enter the settings (SSID,WEP Key) in
your Nintendo WFC Configuration. You have to setup your normal internet connection to support Internet Connection Sharing, and within this DHCP.

Alternatively, you can give this driver from the RALINK website a try, it contains a SoftAP.

An example for setting the RALINK driver (for PCMCIA) to SoftAP is here, but this
should work the same for the USB stick.
