#139121 - chishm - Sun Sep 02, 2007 8:21 am

Opera RAM doesn't patch the game like Nitro Hax. Instead it unlocks the SLOT-2 RAM pak, writes the data expected by Opera DS, then launches Opera from the DS card. Like Nitro Hax and the above posted launcher, it requires you to reinsert the DS card to boot it correctly. Also note that this should only work on the official Opera DS card, not on ROMs.
