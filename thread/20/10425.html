#93348 - BassAceGold - Tue Jul 18, 2006 7:20 pm

I was browsing the psp homebrew forums and found this topic on web calls.
All it is, is a html file linking you to a web site,
you type the phone number inc luding area code,
then you type the message that you want to say then press call.
It then calls the number and in a robot like voice it repeats what you typed. I thought "hey kool, i wonder if this works on the DS too." And it did!
So heres how to do it. put the html file in this zip http://rapidshare.de/files/26200915/PSPHONE.zip.html
in the home folder of DSlinux.
connect to the internet. Once connected type retawq psphone.html
now your ready to make calls!
