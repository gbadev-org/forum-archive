#103513 - Firon - Fri Sep 22, 2006 3:48 pm

If you want to do your own encodes, use BatchDPG, it works much better than DPGTools.

http://ls5.cydonianknight.com/
I've made a custom version of it provides smaller filesizes, if you'd like to use it. http://utorrent.com/BatchDPG-newkvcd.exe
Stick it into your BatchDPG folder (which means install it from ls5's site first!) and run.

Just don't use the VeryHigh, Experimental, or ExpInsane profiles, those aren't tested yet. Same for QNS.

For the audio, you can only use 32KHz if you choose to use (joint) stereo. 48KHz can be used with mono, and sounds better imo.

http://gbatmw.net/index.php?topic=398.60 For further info.
