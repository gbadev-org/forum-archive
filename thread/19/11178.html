the short answer: unless you are able to write it yourself, it probably wont happen.

the long answer: the code to launch a gba rom is specific to the flashcart. This is why you don't see many 'menu' type apps outside of the ones provided by the flashcart makers. Most flashcart makers are NOT cooperative with homebrewers, as they mostly only care about piracy, and thus they do not release any code on how to control their cart.
