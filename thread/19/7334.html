#59405 - Lynx - Tue Nov 01, 2005 4:28 pm

PassMe2 is the new name for the method Olimar came up with for the iQue DS because it had new firmware. (searching the forums would have found a lot of information about this). This method requires the PassMe2 code to be DS cart specific, and it requires SRAM data on your media in the GBA slot. Also, this method currently works on ALL DSs, new firmware or old.

Edit: Oh, stu879, thanks for testing it for me.
