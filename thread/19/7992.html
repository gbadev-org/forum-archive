#65636 - pepsiman - Wed Jan 04, 2006 4:17 pm

WifiMe is a PassMe alternative, that allows DS code to be run from the GBA slot.

WifiMe is sent to the DS using WMB (Wireless MultiBoot).

WMB can also be used to send DS code to the DS.

The DS will reject unsigned code sent by WMB unless you install FlashMe first.

Installing FlashMe requires some sort of GBA flash card and either PassMe or WifiMe.
