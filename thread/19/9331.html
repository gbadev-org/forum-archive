#80542 - caitsith2 - Sun Apr 23, 2006 8:57 am

IIRC, loopy added that feature into noflashme.nds, such that if the firmware version is not known, it defaults to version 3. If the firmware version CRC is known though, it will default back to that version. Pushing a certain button will allow you to change which firmware version is installed. It would not however be a good idea to install anything greater than version 3, if you only have a passme 1. Also, IIRC, version 6 is what DS lite has. Unless the latest version of flashme identified your DS as a DS lite, (via PM register 4, bit 6 == 1), it would not be a good idea to install version 6 onto it.
