Hey, it worked for me.

To clarify:

I was at work copying to stuff to the R4. My work PC is a linux system. I was trying to get skins working and somewhere along the way the R4 started presenting the issue you described. I paniced for a little, then used dosfsck (A linux-only tool) and mkdosfs (Another linux-only tool) to format the card and fix any errors on it.

Using windows, you should probably do a right-click, Format on the card and then run Scan-Disk on it afterwards just to be sure, before re-copying all files to the card.

Worked for me...
