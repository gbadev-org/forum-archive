KidBomba wrote:
Hello, I got my g6 all working and now I can run homebrew! yay! both nds and gba :)

I noticed there is a small button on the cart, does anyone have any idea what this is?

It's for when a program doesn't start and you just get 2 black screens.
It stops the RAM in the G6 from being wiped when you reboot the DS.

2 black screens -> press and hold button -> turn off ds -> turn on ds -> release button.

Homebrew can avoid the need to use this button by setting POWER_CR to turn the screens on.
