#60317 - curiousmike - Wed Nov 09, 2005 3:29 am

Find your copy of usbcable.exe

1. Open command prompt ( Start->Run->Cmd )
2. cd \folder where usbcable.exe resides
3. usbcable -f --- format entire card
4. usbcable -p romname.nds.gba


I don't know if you need to patch roms for passme2.
I do.
