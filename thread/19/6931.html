I want to put a file that finish with .nds on my flash card (x-rom) but my software can only put .gba files.

I bought the superpass DS (passme).

Is there a software that convert NDS files to .gba ?

Sorry if it has been has before, I used the "search" but I didn't find anything...

#54733 - Alec - Thu Sep 22, 2005 12:48 am

http://ds.gcdev.com/dsfirmware/ndsloader.bin

Download that, then put it in the same folder as the nds file you have. For example, we'll use meteosdemo.nds. Use this DOS command.

copy /b ndsloader.bin+meteosdemo.nds meteosdemo.nds.gba

Then flash the meteosdemo.nds.gba to your card.
