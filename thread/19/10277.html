#91675 - tepples - Sat Jul 08, 2006 5:47 pm

1P's DS has FlashMe, a DS Game Card in SLOT-1, and a CF adapter in SLOT-2.
2P's DS has FlashMe, an empty SLOT-1, and a CF adapter in SLOT-2.

To do multiplayer between the systems using an authentic DS Game Card:
  1. On 1P's DS, Hold Select to skip FlashMe, and start the DS Game Card.
  2. Choose a menu option for DS Download Play. Names of menu options will vary depending on title.
    Meteos:
    Mario Kart DS: Multiplayer > Create Group > Simple
    Tetris DS: Multiplayer > Invite > Choose game mode
  3. On 2P's DS, hold Select to skip FlashMe, and choose DS Download Play.
  4. Wait for a game option, choose the game from the list, and press A to confirm your choice.
  5. When 2P's DS appears on 1P's screen, choose "Launch" or "Cut Off" or whatever the game calls it.

To do multiplayer between the systems in a homebrew game:
  1. Verify that the game supports DS to DS multiplayer.
  2. Get within range of an access point that is powered on. The homebrew Wi-Fi library does not yet support ad-hoc or "Ni-Fi" mode and cannot act as a soft access point; you will need an access point, but the access point doesn't need to be connected to the Internet.
  3. If your access point uses MAC whitelisting, look up each DS's MAC address using a commercial DS Game Card: Wi-Fi Settings > Options > System Information.
  4. Start the game on both systems.
  5. Follow the game's instructions.
