DS Flash Equipment > Tip: Send files with WMB directly from the file browser

#67810 - melw - Fri Jan 20, 2006 3:27 pm

Well, this is rather a basic Windows tip than all WMB related, but I thought sharing this might be useful to someone else, too. In short:

- Create a shortcut under your document and settings/<username>/send to/ folder with a desired name (e.g. Nintendo DS).
- Set target to <path to wmb directory>/wmb.exe -data
- Set icon as desired (for example here's a nice one by SgStair: http://www.akkit.org/favicon.ico)

As an end result you can now right click in the explorer a .nds file and choose "send to / Nintendo DS". Simple as that. :)
