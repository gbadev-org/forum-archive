#55657 - Sektor - Fri Sep 30, 2005 9:04 am

It only modifies the header to make it read from the GBA port. The header does not need a valid sig. You need a flash cart/card to run homebrew using WifiMe.

#55677 - tepples - Fri Sep 30, 2005 3:53 pm

Or if you can't afford a flash cart, and you're not interested in playing .gba games, then get a GBA Movie Player and a cheap CF card. It'll only run .mb.gba and .nds files, but many of the ds.gba games are being ported to work in .nds+CF mode.
