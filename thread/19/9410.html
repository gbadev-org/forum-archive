XW Card is short for eXecute/Write Card.

Datel has made a Max Media Launcher, I think it would be useful to make a similar device: It is same as a normal DS game card, but has a switch on top that says EXECUTE and WRITE. When the switch is set to EXECUTE it runs software stored in the card normally. When set to WRITE, the display might say something like "To continue, turn off DS and set switch on DS game card to EXECUTE, then turn on DS". If you press the right combination of button it will display a menu for uploading software to the card which will run in EXECUTE mode. The menu might have options such as this:
Code:
==XW CARD CONFIG MENU==
*DS CARD INFO
 DS UNIT INFO
 COPY SOFTWARE TO XW CARD
 COPY SOFTWARE FROM XW CARD
 COPY SAVEFILE TO XW CARD
 COPY SAVEFILE FROM XW CARD
 EDIT HEX DATA
 WIFI CONFIG
 RUN GBA IN DS MODE
 POWER OFF

Code:
==COPY SOFTWARE TO XW CARD==
*CF CARD
 AUDIO
 HTTP
 FTP
 RAW TCP OR UDP DATA
 DS DOWNLOAD PLAY
 RETURN TO MAIN


Then you could run homebrew software on Nintendo DS without any extra equipment! You could give card to whoever you want to, they can use the card just fine.

You still need some equipment to make DS game, such as Supercard CF, wireless router, etc. Or you could type it manually in (but that takes too long).

Also, if you give card to someone else and they change it, you can put it back how it was before.
