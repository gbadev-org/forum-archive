#119464 - tepples - Fri Feb 23, 2007 5:55 am

A SLOT-1 adapter such as R4 will let you run DS programs.

A SLOT-2 card such as SuperCard, CycloDS, or M3 will let you run GBA programs. A SLOT-2 card plus a NoPass card (SuperKey, CycloDS Key, PassCard 3, MK4-mini, MAX Media Launcher) will let you run both GBA programs and DS programs.

These adapters take SD, miniSD, or microSD memory depending on the model. You will need appropriate memory cards and a USB memory card writer.
