#109478 - KoshNi - Sat Nov 18, 2006 4:26 pm

When you can't run a .nds file on a Supercard, first try to patch it with this
http://l33t.spod.org/ratx/DS/SC/scmhbp/scmhbp-0.2.zip (scmhbp.exe) in order to create a .sc.nds file : simply drag the .nds file over the scmhbp.exe icon and the .sc.nds file will appear in the directory where the .nds is stored. Put the .sc.nds on your SD card and try to run it from the Supercard boot menu

The case of Moonshell a bit more complicated because it reads files and every device hasn't the same settings... What you can try is to get this version http://home.att.ne.jp/blue/moonlight/moonshell141_with_dpgtools11.zip , unzip it then run setup.exe . When the configuration window open, first select the directory where you want Moonshell to be installed (=the letter corresponding to your SD card) and then look at the right part of the window, entitled ROM image. Unselect every checkbox excepted the one corresponding to your device type (= Supercard SD), then click the setup button. Now it should be working fine...
