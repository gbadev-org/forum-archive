#45603 - FourScience - Sun Jun 12, 2005 5:25 pm

Well that is IF he patched the firmware, he can run homebrew .nds files using wmb.exe.

.ds.gba is for GBA flash carts.
.nds is for DS Download Play using wmb.exe. The commercial demos will run without the firmware patch, but the homebrew demos require the firmware patches, as they remove the check for the RSA signature.

Read the other threads for more info on wmb.exe and how .ds.gba ROMs can be built from .nds ROMs.
