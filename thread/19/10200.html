From the PIRACY-DOT-COM Wiki:

Quote:
FlashMe is a collaboration between Loopy, FireFly, and DarkFader. It's a hacked firmware for DS that allows you to start code in DS mode from a flash card in the GBA slot.

Quote:
PassMe type devices are inserted into the DS slot and tricks the DS into allowing unsigned DS code to run from the GBA cartridge slot in full DS mode.

Quote:
NoPass type devices are inserted into the DS slot and trick the DS into allowing unsigned DS code to run from the GBA cartridge slot in full DS mode.

NoPass is the third generation of PassMe type devices. The name "NoPass" was tagged by the head developer of DSLinux, PepsiMan. They're the same size as an original DS game card. They work completely stand-alone and do not require an original DS game or any kind of firmware programming and work with any type of DS with any firmware, including DS Lite.
