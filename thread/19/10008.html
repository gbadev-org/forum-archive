#89026 - Ryan FB - Thu Jun 22, 2006 9:09 pm

Moonshell's source has the source code for its EZ SD driver, but the FAT libs in Moonshell are heavily modified so integrating that back into another FAT lib version isn't straightforward. I've tried building PicoDriveDS with EZ SD support enabled, but it hangs on boot on my EZ4 Lite microSD. So you might have to wait until one of the other FAT lib authors merges that code back in before you see it start to trickle down to the homebrew that uses it.
