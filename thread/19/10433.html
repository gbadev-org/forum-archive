What you want is Darkain's MultiNDS loader. You don't see much talk about it, because most people at this point use CF or SD based cartridges.
You can get it here: http://www.darkain.com/ds/hack/ -- I think you want "Final Multi-NDS Loader (includes teh everythings)"

There is zero support for using GBAMP-SD in DS mode currently, sorry. If it's like the GBAMP-CF, you can load small GBA multiboot(mb) homebrew, but that's about it.
