#154670 - Sektor - Sun Apr 20, 2008 3:55 am

You can use it as a method of sending from a PC to the DS. If you are running Windows then you need a PCI/Cardbus wi-fi card with a RT2560 chipset. Install the driver and WMB Application and then run WMB.exe -data file.nds.

Boot the DS, select "Download Play" and wait for the game to show up.

does the router thing still function as a normal one too? Or would I just be better off using my card normally?

#154674 - Sektor - Sun Apr 20, 2008 5:42 am

ralink cards can only be used for WMB when Firefly's driver is enabled. You have to switch back to normal driver if you want to use it as a normal wi-fi card. Firefly's driver doesn't support USB ralink cards, so you can't use the Nintendo Wi-fi connector.
