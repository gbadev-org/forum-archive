#82987 - 0xtob - Thu May 11, 2006 6:24 pm

I noticed that you confused WMB and Wifime here:

WMB=Program to transfer homebrew/demos to your DS and run them
WiFiMe=File that you transfer to your DS via WMB to run code from a Flashcart or a CF/SD adapter card.

To transfer homebrew to your DS via WMB, you need to install FlashMe. To install Flashme, you need a Flashcart or a CF/SD adapter card to run the installer from.
