#69171 - Xtreme - Sat Jan 28, 2006 11:54 pm

Confirmed to work with the latest G6 4.0b + firmware and M3 V22b + firmware.

Hi.
I have G6 flash cart and I'm lazy. So I made this command-line script for myself, but later time thinked that I should share it. I hate to do things manually. EDIT: Now I also have M3 Adapter.

What is this?
G6
It makes backups of those *1.0 (DEFAULT) saves and puts them in folder named with the current date (DD.MM.YYYY, that's the current order.. maybe later I will make it to use ISO date format).
Does not make backups of 1.1 or 1.2 files (not needed, I think).
It does not do any harm on your computer. It does have some basic checks to prevent any faults (I like to make sure that everything works).
M3
It backups those DAT files in NDSSAVE folder.

How to use it?
You can simply download the ZIP file and UNZIP it. Plug in your G6/M3 and run the G6 or M3 script file and it will backup the saves if they are found.

It's 100% working with Windows XP. Not tested with other systems. You can run it in any folder (in your Windows PC, so it supports long file names).


And last: Do not criticize too much my code if you don't have better solution. I'm not a coder. Only able to write scripts & html.

EDIT: G6 script was updated from 0.03 to 0.04. M3 0.01 script added. 7-Zip exe file renewed.
