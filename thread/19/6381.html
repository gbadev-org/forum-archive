One last questions.Is there a program that turns a .nds into a .nds.gba?Thanks


For that you need Darkfader's loader, available here.

Look under Development Tools for ndsmall.bin. Usage is thus (open DOS console for this):

copy /b ndsmall.bin + demo.nds demo.ds.gba

Basically this just copies the ndsmall.bin into the front of the NDS...then passme loads that loader which then loads the NDS code from the NDS file...heh, sounds confusing when I say it this way.
