#139278 - tepples - Tue Sep 04, 2007 5:10 am

Something else that might work (not tested because I don't use wmb, though I do use the Windows command prompt):

Create a file in the same folder as wmb.exe. Call it wmbsend.bat (not wmbsend.bat.txt). Its icon should resemble a little window with a gear in it. Open the file in Notepad and put this in it:
Code:
wmb -data %1
pause

Then drag .nds files on top of this batch file.
