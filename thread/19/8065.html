#66634 - MaHe - Thu Jan 12, 2006 3:57 pm

1. You must disable the loader. Somehow.
2. Put FlashME.nds and ndsmall.bin in the same folder (eg. C:\DS\)
3. Use this command: copy /b ndsmall.bin + flashme.nds FlashME.ds.gba
4. Burn the flashme.ds.gba file on a FlashCart and DISABLE the loader.
