altometer wrote:
okay i have a gbamp. and a passme is coming in the mail. what do i do? i want to be able to play homebrew, are there anymulti loaders out there that work for ds? and what do i have to do to my gbamp to make it work wiht this stuff


Ok, do you have the new firmware? If you do, you can't use the GBAMP to boot the NDS using PassMe2 since it doesn't have SRAM.

Anyway, assuming everything is in order, you need to get Chism's GBAMP firmware hack v2 first of all. (found at http://www.geocities.com/chishm1/gbamp/index.html). You need to boot the GBAMP up as normal, and then go into the NES Game option and launch NDSMP.gba. Follow the instructions there to update the firmware of your GBAMP (you may need to install an official firmware update for the GBAMP after doing this...old versions required you to do so but I don't think the newer ones do).

After that, when you get your PassMe, put an NDS file on your GBMP in the root directory named _BOOT_MP.nds. This can be any NDS homebrew, I recommend a loader or something with a loader built in, such as MightyMax's GBAMP Multiboot or MoonShell. From there just start up the NDS with the PassMe and the GBAMP inserted and it will boot up the _BOOT_MP.nds file on the card. :)
