#78789 - tepples - Sun Apr 09, 2006 8:09 pm

If a game is not listed on "warez scene" release lists at PIRACY-DOT-COM and PIRACY-DOT-NET:
The game has not yet been released in the warez scene, or it was released after the release list's current maintainer went AWOL.

If a game is listed on the scene release lists but not listed in darkfader's table:
It's too new and has not yet been analyzed with the PassMe2 search tool. This has to be done by someone with a v1-v3 firmware DS, a traditional GBA flash card, an original PassMe, and the game in question.

If a game is listed in darkfader's table but PassMe CPLD and PassMe SRAM columns are empty:
There have to be specific 16-bit values in the first .nds file on the card in order for PassMe2 to work. Many games don't contain those values.
