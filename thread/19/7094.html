#56477 - Lynx - Sat Oct 08, 2005 10:46 pm

Uh.. I wouldn't do that if I were you... There is no new Firmware update available for the iQue (that I know of)... It is totally different.. The iQue DS PassMe code allows you to run homebrew.. Untill someone dumps the firmware and patches it, you will be stuck using PassMe/WiFiMe.

As for the Super PassKey and other "commercialized" versions, none of them are strait forward reprogrammable. I haven't looked, but some might even be "write" protected. Though, with the CPLD we are using, that doesn't really mean anything, as DF researched, and there is a way to overide that function. But, the Super PassKey is glued together, so opening one would be a pain in the butt. The Magic Key on the other hand, opens very easy.. You'd still have to figure out the JTAG programming points.. but it could be done.
