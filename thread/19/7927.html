wildex999 wrote:
Can someone tell me that this combination will work?

-EZ-Flash III (2G) Flash Set

You can better get the EZ-Flash II 512Mb or II 1Gb because it has more SRAM save needed for some games. For homebrew the EZ-Flash 128Mb is good enough. If you have a newer Nintendo DS you also need a passme2 or EZ-Pass2 with the right code written on it. (You need to stick a supported original Nintendo cartridge in the back before it works).
