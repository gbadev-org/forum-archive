#89305 - tepples - Sat Jun 24, 2006 3:17 am

When you double-click a Windows command-line program, it prints a message stating that one or more arguments were missing and then stop. When the program stops, Windows closes its window, hiding the message. To prevent this from happening, open the GBA beginners' FAQ and search for the answer to the question:

Q: I've downloaded a program, but when I double-click its icon in Windows Explorer, a window filled with black opens and closes so quickly that I cannot read it. How do I use this program?
