Update.e19 English version 2.0(2005.9.1)

1.Long file name support
2.Support <200k GB and GBC format games. You need to put Gooba.gba file to the root directory of CF so that you can run those games.
3.Available music modes: All,Repeat All,Repeat One,Single,Random,Rev Repeat.
4.Sleep modes: a)When the time is set, GBA/GBA SP/DS will be in sleep mode without pressing any button. b)GBA/GBA SP/DS will change to sleep mode when the music is finished. Note: Only when the music mode is "all" or "single", then the sleep mode works.
5.Auto Page: Set the time, then it will turn the page automatically. press "select" botton to make the setting.
6.Support 256 backing files: Save the backing files in a folder named"BACKING". All backing files must be named as BACKING*.***.
7..Password: If you run this function, type the password before entering the GBA MP menu. To cancel it, just delete all files from your CF card.
8. Ebook scroll bar is available.[/url]
