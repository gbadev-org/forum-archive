#76423 - lophyte - Tue Mar 21, 2006 4:39 pm

Does Chishm's GBAMP firmware hack essentially turn it into a GBA flash cart? I suppose what I'm really asking is, if I FlashMe using my GBAMP+Chishm's firmware, can I also recover if FlashMe fails (or if the NDS firmware gets corrupt) using the GBAMP, in the same way you'd use a GBA flash cart?

#76710 - Lynx - Thu Mar 23, 2006 8:23 pm

I believe the answer is no and yes.

No, it doesn't "turn" it into a flash cart, but it does make it bootable from a PassMe.

Yes, you should be able to recover using it by renameing flashme.nds to _BOOT_MP.nds. But, I haven't tried it myself.

#76802 - HyperHacker - Fri Mar 24, 2006 10:34 pm

The hack allows you to boot NDS multiboot (homebrew/demo) programs from it with a PassMe or FlashMe, loading them into memory. It can't run GBA ROMs like a flash cart can, and won't work with PassMe2.

#77728 - Mighty Max - Sun Apr 02, 2006 6:12 pm

You will still be able to use it normally.

It will boot into the normal menu
- when you overrided the FlashMe Autostart via keypresses
- when no _boot_mp.nds file is present on the flashcard
- when a .nds sets the NDS into gba mode
