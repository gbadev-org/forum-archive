I have recently opened up my Max Media Dock (MMD) for NDS . Following is my finding.

When you open up the MMD you will find only 2 major chip in this .

1) cypress CY7C68300B USD to ATA bridge
2) ST EEPROM

It has no SRAM no Latches No Bios like other SC and Easy Flash . What does it means? It measn that it cannot be used with Flashme currently as it does not emulate a GBA cart at all and it cannot be use to run the GBA code as it would be to slow to run from the CF but is it fine for DS code

This is how i suspect the MML + MMD works ( correct me if i am wrong please)

the Datel media player or media launch works buy emulating the NDS catridge . Unlike traditional Passthru device that jump the address to the GBA flash address it actually hold a NDS program inside it whhich access the MMD CF and load a default NDS code ( in this case Bootme.nds)

The Bootme.nds is a browser software which inturn load MP3 Files or NDS files from the CF.

That how i figure how MMD works please correct me if i am wrong
