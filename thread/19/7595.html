#61935 - pepsiman - Fri Nov 25, 2005 3:14 pm

pimpmastah wrote:
First problem I got a blue DS with the new firmware, got a passme2 for Mario 64 DS EUR and a flashcard with flashme(v5) installed, but when I start up the DS I only get two white screens. What?s wrong?

As you don't mention writing the SRAM, I'm going to assume that the problem is that you haven't written the SRAM.

This program will write the SRAM for you: http://pepsiman.pwp.blueyonder.co.uk/passme2flashme.gba

Run that in GBA mode, select ASMP, insert passme2 and reboot.
FlashMe will then start.
