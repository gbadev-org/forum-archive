Any ideas?

#73944 - wbo - Wed Mar 01, 2006 2:27 am

I have tried several times to navigate the M3 menu blindly when using the Flashme recovery code. I think that the M3 interface code must be crashing when it is run using the Flashme recovery code.

I am using firmware version E16 on my M3. Has anyone else gotton the M3 to work under the Flashme recovery code? If so, what firmware version are you using on your M3?

EDIT: The Flashme recovery code does work just fine on the M3 as long as you do not have a DS game in slot 1. When I was testing I had Animal Crossing inserted in slot 1 and all I got was blank screens. When I took Animal Crossing out, the Flashme recovery code booted into the M3 menu just fine.

So if anyone else has this problem, just remove any games from slot 1 and it should work fine.
