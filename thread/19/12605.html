#120352 - dantheman - Sat Mar 03, 2007 3:17 am

Reaper, you're probably going to get better help at the DSLinux forums. Like we've attempted to tell you, reading the DSLinux wiki would help greatly. It says right there that
Quote:
If you have a Max Media Drive some people have reported that the GBAMP version may work


So go to the downloads and select the GBAMP version to download. Extract the file using a program like 7zip. This is very important, as you cannot simply place the *.tgz file on your card. It won't work. Fully extract it and you should get a dslinux.nds file and a folder called "linux". Place the "linux" folder in the root directory of your card, and place dslinux.nds wherever. If the MMD has a specified Games folder, you might want to put it there. Launch the MMD, load the dslinux.nds file, and then read here to find instructions for setting up your wireless network. Only once you setup your wireless network will Retawq work.

Extract that resulting file as well. From my understanding, a .tgz is actually a .tar.gz file, and it was compressed with two different algorithms. Therefore, you need to extract it once and then extract the resulting file. Once you're done, go into the linux folder and make sure there is a "Home" directory. If there isn't, then you need a different extraction program that doesn't remove empty folders.
