#62182 - tepples - Mon Nov 28, 2005 3:07 am

First of all, you'll need a GBA card, which is either a CF adapter with 32 MB of RAM (M3 or SuperCard) or a NOR flash card (such as F2A, EFA, or XG). Make sure that it can run GBA homebrew before going further.

Then you can install FlashMe, the modified DS firmware, using one of the second generation modding methods. Doing the ppflash method is reportedly as hard as soldering on a modchip, but using the PassMe2 adapter is much easier.
