#88334 - tepples - Sun Jun 18, 2006 9:35 pm

As of 2Q 2006, NoPass is the newest method to boot homebrew on the Nintendo DS. It comes in the form of a card shaped like a DS Game Card, and it works by emulating the encryption of an authentic DS Game Card to get into DS mode, but then jumping to DS code on the GBA card.

Like PassMe original, NoPass works with GBA Movie Player v2 CF. (PassMe2 doesn't, as it needs GBA SRAM.) Like PassMe2, it works on newer versions of Nintendo DS firmware. (PassMe original needs old firmware.) Unlike PassMe original and PassMe2, it does not require an original DS game and can be shrunk to fit in the same size as a DS Game Card.

NoPass devices include MAX Media Launcher, SuperKey, PassCard 3, and MK4-mini. Check Retailer Feedback for stores near you.

Compatible GBA cards:
  • GBA Movie Player v2 CF with Chishm's NDSMP
  • GBA Movie Player v2 SD with latest firmware
  • SuperCard (all versions)
  • M3 (GBA Movie Player v3) (all versions)
  • SuperCard SD
  • Flash2Advance Classic

Instructions:
  1. Insert the NoPass device in SLOT-1.
  2. Insert the GBA card in SLOT-2.
  3. Power on the DS.


What about PassMe2?
If you are buying DS homebrew equipment for the first time, we recommend buying a NoPass. If you have already bought a PassMe2, see this topic:
New Firmware? Look Here for PassMe2 info

Where can I learn about other boot methods?
I maintain a page about various DS homebrew boot methods, giving their history and a bit of speculation.

Do I need a NoPass or PassMe or FlashMe to use GBA homebrew on my DS?
No. If you don't use a DS boot method, the DS acts like a GBA without a Game Link port.
