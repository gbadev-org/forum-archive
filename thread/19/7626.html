#62188 - tepples - Mon Nov 28, 2005 3:31 am

paco1234 wrote:
on the page http://darkfader.net/ds/ i have seen the schematic to connect a normal CF-card to the NDS.

An adapter that includes a circuit nearly identical to DarkFader's adapter is mass-produced, called the GBA Movie Player. However, the GBA Movie Player currently cannot function in DS mode on a Nintendo DS system with new firmware until the DS has been modded, and you can't use the GBAMP to mod the DS either.

Quote:
Is it possible to run homebrew programs on the DS using a CF-card and wifime ?

Yes, but if you're using WiFiMe or a GBAMP, your programs can use only GBA mode unless your DS either has old firmware or has already been modded.

Quote:
Is it possible to store the data on the CF-Card using a normal usb-CF-card reader/writer ?

Yes.

Quote:
I have got the "new" nintendogs-bundle-blue DS. and the mariocart-bundle-silver DS. I can solder directly to the connector ...

If you have a DS that was bundled with any game other than First Hunt, then ten bucks says you have the new firmware that does not work with the GBAMP or with WiFiMe.
