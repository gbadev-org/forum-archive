#96400 - tepples - Thu Aug 03, 2006 2:26 am

HyperHacker wrote:
If it is flashable, you should be able to rewrite it using the DS alone.

Unless it is flashable only using voltages that the DS cannot produce. Several rewritable memory parts (EPROM, EEPROM, or flash) use 3.3 V or 5.0 V to read and 12.0 V to write.

Or unless it is flashable only through connectors that the DS doesn't have. For instance, the PassMe2 can be flashed only through its JTAG connector, and the EFA-Linker (Extreme Flash Advance) can be flashed only through its USB port.
