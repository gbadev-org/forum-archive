Does your card at least say "ENLWI-G-RLAM" somewhere? If not, then they sent you the wrong item and you should request that they send the correct one. If it does, see if you can find any Revision numbers.

Also, download PCI Tree (http://www.pcitree.de/), unzip it, follow the instructions and place HLP.SYS in the correct folder. Run it, and look down the left window list, under PCI/PCI Bridge (or something like it), and see if you can find anything with Ralink under there. If you can't find Ralink, select whatever has Network Controller in it as long as it's not another network card you have installed. Once it's selected, look on the top right area and report back the VID and DID descriptions. DID should be 0201 and VID should be 1814. These same instructions are on the WiFiMe tutorial page, with a screenshot of PCI Tree also (http://www.aaronrogers.com/nintendods/wifime/images/pcitree_01.png).
