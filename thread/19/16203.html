#164618 - Sektor - Mon Nov 10, 2008 4:34 am

If you hold select on powerup, it should go to the DS firmware menu. You can set the DS to automatic or manual slot-1 booting. Some cards need to be started automatically and they overide the DS setting (unless you hold select). The button combinations can vary if you have flashme installed (should still be some combo of select and/or start).
