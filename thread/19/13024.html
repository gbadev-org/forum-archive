#131305 - Sweater Fish Deluxe - Thu Jun 14, 2007 2:28 am

There's an INI in the DSYSTEM folder that as an autoboot option which you can turn on and off. The NDS to be autobooted also has to be in the DSYSTEM folder and it should be named AUTORUN.BIN (despite the BIN extension it should be a regular NDS file).

I haven't tested this, so I don't know how it works. I don't actually have one of these cards, so I haven't been able to try it. I was looking forward to getting NeoFlash's version when I made my last post, which is why I'd been looking into the card's features, but the card I was supposed to get from them was lost in the mail and NeoFlash refused to send me a replacement (which seems totally unfair considering the card was a prize I won for my first ever coding effort on the DS and they'll give free ones for review to any monkey with a domain anyway, but...well...I digress).

Let me know how it works if you try it. I know that the R4's autoboot feature is sort of lame since it actually loads the regular menu and then loads the AUTOBOOT.NDS file from there rather than loading the autoboot file in place of the GUI.
