#58648 - tepples - Tue Oct 25, 2005 1:51 am

  1. Open a command prompt
  2. cd to the folder containing ndsloader.bin and flashme.nds
  3. copy /b ndsloader.bin+flashme.nds flashme.nds.gba
  4. Write flashme.nds.gba to the card.
