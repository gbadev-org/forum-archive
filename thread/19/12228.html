This is certainly a valid concern. FlashMe's built-in recovery code is incompatible with slot 1 devices, so if your firmware got corrupted, you'd have to buy a slot 2 device to fix it. :-/ I do wonder though if there's even any chance of fitting slot 1 boot code into the failsafe region. Apparently FlashMe's already pretty stuffed, and the code would probably be more complicated than slot 2 (so even making a version supporting only slot 1 may not work); since they emulate/are a DS card, you'd need to do authentication and all that fun.

I might just try to write my own custom firmware, there's a few things about FlashMe that bug me, such as forcing the backlights on. (Not to mention the legal issues.)
