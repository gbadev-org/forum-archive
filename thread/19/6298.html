Ok...the nds.gba file is not really a GBA file. It's an NDS format file with a loader attatched to the front of it (that also has a GBA header just for the flash cart software mostly). The code is NOT GBA compatible, it's meant for the Nintendo DS.

I don't think you can use a stock GBMP with PassMe (it isn't going to know how to load something from the Compact Flash card)....Dark Fader's work on cracking the GBMP's firmware may change this though, so if you really are hell bent on using the GBMP wait and see. :)
