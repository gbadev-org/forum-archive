simple: you MUST rename the flashme.ds.gba to flashme.nds before putting it on the CF card. I can almost guarantee it will work then (provided there isnt something else holding you back like booting into the SuperCard DS mode...).

as to the firmware of the SuperCard, hold L and R shoulders while the gba logo is onscreen as you are booting the SuperCard. Pressing A will cycle through the test menus and one of the screens has a version number on it.

That 1.20 thing is a bug, apparently it was fixed but the version of the download on the SuperCard site never changed...

#52262 - look - Sun Aug 28, 2005 10:53 pm

I flashed my Nintendo DS firmware using wifi and supercard CF

Here's what I did

PREPARATION

Get a compatible WIFI PCI card

I used the ?17 sitecom wl-115 card from play.com:

http://play.com/play247.asp?pa=sr&page=title&r=ELEC&title=602457

Use this driver:

http://users.belgacom.net/bn967347/download/rt2560_driver_1_0_0_8.zip

Use this version of wmb.exe:

http://www.aaronrogers.com/nintendods/wifime/rt2560_app_wmb_1_1_fixed.zip


Download these files:

http://ds.gcdev.com/dsfirmware/ndsloader.bin

http://ds.gcdev.com/dsfirmware/flashme.nds

Run this command in windows command prompt in the dir containing the files

copy /b ndsloader.bin + flashme.nds flashme1.nds

copy flashme1.nds to your CF

FLASHING

start wmb.exe on the machine with the compatible wificard/driver on it

boot the DS and download the wireless multiboot program

The supercard menu should now pop up

run flashme1.nds

Follow the instructions on how to flash the firmware at:

http://www.dspassme.com/FlashMe.shtml
