#85017 - HyperHacker - Fri May 26, 2006 1:01 pm

You should have just posted this in your other thread. Official demos have an RSA signature, which is like a checksum (to be specific, it's an RSA-signed checksum) but encrypted. Adding the signature to a program requires Nintendo's key, and as far as we know they're the only ones who have it (and cracking it would take a couple hundred years). The DS will refuse to play demos that aren't signed, which means if you send homebrew or a modified demo it will do exactly what you described. (WifiMe is actually an official demo, but parts of the file that aren't RSA-protected are modified.) If you want to send homebrew via WMB you need to install FlashMe.

BTW, WifiMe = wireless PassMe; Wireless Multiboot (WMB) = sending demos/homebrew to Download Play.
