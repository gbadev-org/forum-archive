#54747 - cybereality - Thu Sep 22, 2005 5:48 am

Killjoy: I had a few issues with the multi-loader when I first got set up, but it works fine. The main problem is that if you attach an invalid file it will sometimes make the whole loader crash. Only use tested homebrew code and you should be fine (ie the examples that come w/ ndslib). Also, make sure you are using the most recent loader (the one with the grey/white screens, not the one packaged with darkstar). I am using a NeoFlash, so you may have a different issue, but I hope it helps.
