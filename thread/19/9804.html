No problem, probably should have explained it in the post. Mostly I just mean this...is it the NDS from the site, directly, with no modification you are flashing to the flash cart? If so, it's possible that your GBA flashcart's software is screwing up the NDS in some way (newer NDS homebrew should work even without a loader if the flashing software isn't messing with them, but you can't be too sure).

Go here to Darkfader's site and download ndsmall.bin, part way down the page in the 'DS Development Tools' section. Then, assuming you are using Windows, open a command window (you can do this by pressing the 'Start' button, picking 'Run' and typing command into the window that pops up).

If you don't know DOS commands, it's probably easiest to put ndsmall.bin and flashme.nds onto your desktop (I know XP's command window starts there, I think it does in 98 + 95 as well). Anyway, type the following:

copy /b ndsmall.bin + /b flashme.nds flashme.ds.gba

and then try flashing that onto your flash cart, see if this fixes it. Alternatively, if you have MoonShell, it comes with a program in the wintools folder called dsbuild that if you drag an NDS onto it it'll create a ds.gba file for you.
