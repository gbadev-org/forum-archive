You don't have to wait for RSA to be cracked (highly unlikely) or for the SM64DS protocol to be cracked (much more likely) to run code over Wi-Fi. Just get a PCI Wi-Fi card with a Ralink chipset and a ?39.95 EFA-LINKER flash card from totalgba.com, put FlashMe on it, and use WiFiMe to boot it. Now you can send any .nds file with WMB, and you can put the flash card away to use with your GBA projects.
