For slot1 only device, get an R4. It's cheaper than M3 Simply but identical hardware.

Slot2 Supercard Lite + Superkey is still a good option, slightly more expensive than an R4 but it can run GBA homebrew, it can be used as a RAM expansion and it can run homebrew that uses an appended file system. Just don't get the rumble version since it has no RAM.
