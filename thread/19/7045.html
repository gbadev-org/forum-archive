#56070 - tepples - Wed Oct 05, 2005 9:34 pm

Galant wrote:
i have to patch every file to nds.gba for making it work, is there no way to run .nds files directly?

Many newer .nds files have a built-in loader that lets them work as both a .nds file (for FlashMe+WMB or for any+GBAMP) and a ds.gba file (for any+GBA flash card). This results in an invalid GBA header, but PassMe, WiFiMe, and FlashMe don't care. However, many GBA flash cards' official writing programs insist on "repairing" the GBA header, which obliterates the loader. For these, you need to prepend a stand-alone loader that doesn't rely on the first 192 bytes of the card. Or look through your program's option screen to see if you can turn off "fix header" or something like that.
