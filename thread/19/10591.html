#95652 - tepples - Sun Jul 30, 2006 6:23 pm

FlashMe patches the DS firmware to do the following before it starts the official firmware:

[ Power on ]
If A+B+Select+Start is held, jump to GBA ROM (failsafe shortcut)
Initialize DS hardware
If A+B+X+Y is held, jump to GBA ROM
If four-character game code in GBA ROM header is "PASS", jump to GBA ROM
Jump to official firmware
