mattfl wrote:
thanks, the bundled version of nintendogs isn't the same as the nintendogs lab and friends that i can use with it is it? Is there a way to find out.

Look at the DS Game Card's label for an alphanumeric code that begins NTR- and then match it to one of the codes listed on the passme2 order form.

Quote:
sram.bin file stored on your Flash Cart

does that mean a specific file stores on my cf card that plugs into the m3 or what?

For the M3, you have to use the instructions that were posted elsewhere to set the sram.bin as the save file for FlashMe.

Quote:
do I need to keep the passme in the Ds at all times?

You can remove it after booting.
