#100954 - tepples - Fri Sep 01, 2006 1:26 am

DS programs for SuperCard need a loader. Anything ds.gba can just be renamed to sc.nds, and a lot of newer .nds files work as in because the newer startup code includes a basic loader. For the older ones built with older versions of devkitARM, try running them through dsbuild:
Code:
dsbuild kitties.nds -o kitties.sc.nds
