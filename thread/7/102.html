#593 - tutifruti - Wed Jan 08, 2003 10:28 pm

Hi! You are doing a comparison, not an assignment:

Change all the '==' :

ScreenMem8[x+(y*32)] == (map[x+(y*16)])*4+1;

to '=':
ScreenMem8[x+(y*32)] = (map[x+(y*16)])*4+1;

Also your code is wrong, video memory is 16bits, not 8bits.
