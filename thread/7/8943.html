#75992 - sgeos - Fri Mar 17, 2006 1:49 pm

return ends the function.

If you want to return at the end, do this:
Code:
int rval;

for (...)
{
  if (...)
    rval = 1;
  if (...)
    rval = 2;
}
return rval;
